## 문제

[BOJ 3041 - N-퍼즐](https://www.acmicpc.net/problem/3041)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
line = ""
for _ in range(4):
    line += input()

distance = 0

ANSWER = "ABCDEFGHIJKLMNO"
for i in range(15):
    for j in range(16):
        if line[j] == ANSWER[i]:
            dx = abs((i % 4) - (j % 4))
            dy = abs((i // 4) - (j // 4))
            distance += dx + dy
            break

print(distance)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
