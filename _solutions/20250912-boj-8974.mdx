## 문제

[BOJ 8974 - 희주의 수학시험](https://www.acmicpc.net/problem/8974)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{

  int a, b;
  cin >> a >> b;

  vector<int> vec;

  for (int i = 1; i <= 1000; i++)
  {
    for (int j = 1; j <= i; j++)
    {
      vec.push_back(i);
    }
  }

  int sum = 0;

  for (int i = a - 1; i <= b - 1; i++)
  {
    sum += vec[i];
  }

  cout << sum;
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
