## 문제

[BOJ 1245 - 농장 관리](https://www.acmicpc.net/problem/1245)

> N×M 격자의 높이 정보가 주어질 때, 봉우리의 개수를 구한다. 봉우리는 같은 높이의 연결 영역 중 인접한 모든 셀이 자신보다 높지 않은 영역이다.

## 풀이

DFS로 같은 높이의 인접 셀을 탐색하면서, 주변에 더 높은 셀이 있으면 봉우리가 아닌 것으로 판정한다. 8방향 탐색을 사용한다.

## 코드

```javascript
const solution = (input) => {
  const [N, M] = input[0].split(" ").map(Number);
  const grid = [];
  for (let i = 1; i <= N; i++) {
    grid.push(input[i].trim().split(" ").map(Number));
  }

  const visited = Array.from({ length: N }, () => Array(M).fill(false));
  const dx = [-1, -1, -1, 0, 0, 1, 1, 1];
  const dy = [-1, 0, 1, -1, 1, -1, 0, 1];

  let isPeak;

  const dfs = (x, y, h) => {
    visited[x][y] = true;
    for (let d = 0; d < 8; d++) {
      const nx = x + dx[d];
      const ny = y + dy[d];
      if (nx < 0 || nx >= N || ny < 0 || ny >= M) continue;
      if (grid[nx][ny] > h) isPeak = false;
      if (grid[nx][ny] === h && !visited[nx][ny]) dfs(nx, ny, h);
    }
  };

  let result = 0;
  for (let i = 0; i < N; i++) {
    for (let j = 0; j < M; j++) {
      if (!visited[i][j]) {
        isPeak = true;
        dfs(i, j, grid[i][j]);
        if (isPeak) result++;
      }
    }
  }
  return result;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N × M)
- 공간: O(N × M)
