## 문제

[BOJ 4539 - 반올림](https://www.acmicpc.net/problem/4539)

> 주어진 수를 일의 자리부터 차례대로 반올림하여 가장 높은 유효 자릿수만 남긴 결과를 출력한다.

## 풀이

일의 자리부터 시작하여 해당 자릿수가 5 이상이면 올림, 미만이면 버림한다. 자릿수를 10배씩 증가시키면서 최상위 자릿수까지 반복하여 처리한다.

## 코드

```cpp
#include <iostream>
using namespace std;
#define endl '\n'
int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);
  int N;
  cin >> N;

  while (N--)
  {
    int X;
    cin >> X;

    for (int i = 10; i < X; i *= 10)
    {
      if (X % i / (i / 10) >= 5)
      {
        X = (X / i + 1) * i;
      }
      else
      {
        X = X / i * i;
      }
    }
    cout << X << endl;
  }
}
```

## 복잡도

- 시간: O(N * d) (d: 자릿수)
- 공간: O(1)
