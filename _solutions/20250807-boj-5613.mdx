## 문제

[BOJ 5613 - 계산기 프로그램](https://www.acmicpc.net/problem/5613)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(NULL);

  int a;
  cin >> a;

  while (true)
  {
    char op;
    cin >> op;

    if (op == '=')
    {
      break;
    }

    int b;
    cin >> b;

    if (op == '+')
    {
      a = a + b;
    }
    else if (op == '-')
    {
      a = a - b;
    }
    else if (op == '*')
    {
      a = a * b;
    }
    else if (op == '/')
    {
      a = a / b;
    }
  }

  cout << a;
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
