## 문제

[BOJ 2929 - 머신 코드](https://www.acmicpc.net/problem/2929)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import re

t = input()
p = re.split("(?=[A-Z])", t)

cnt = 0
for i in range(1, len(p) - 1):
    a = len(p[i]) % 4
    if a != 0:
        cnt += 4 - a
print(cnt)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
