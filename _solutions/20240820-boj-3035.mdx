## 문제

[BOJ 3035 - 스캐너](https://www.acmicpc.net/problem/3035)

> R x C 크기의 문자 패턴을 행 방향 ZR배, 열 방향 ZC배로 확대하여 출력한다.

## 풀이

각 행의 문자를 읽으면서 각 문자를 ZC번 반복하여 확대된 행 문자열을 만든다. 그리고 해당 행을 ZR번 반복 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
  int r, c, zr, zc;
  cin >> r >> c >> zr >> zc;

  for (int k = 0; k < r; k++)
  {
    string str;
    for (int i = 0; i < c; i++)
    {
      char ch;
      cin >> ch;
      for (int j = 0; j < zc; j++)
      {
        str += ch;
      }
    }
    for (int i = 0; i < zr; i++)
    {
      cout << str << endl;
    }
  }
}
```

## 복잡도

- 시간: O(R * C * ZR * ZC)
- 공간: O(C * ZC)
