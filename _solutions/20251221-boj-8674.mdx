## 문제

[BOJ 8674 - Tabliczka](https://www.acmicpc.net/problem/8674)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
  long long int a, b;
  cin >> a >> b;

  if (a % 2 == 0 || b % 2 == 0)
  {
    cout << 0 << '\n';
  }
  else
  {
    cout << min(a, b) << '\n';
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
