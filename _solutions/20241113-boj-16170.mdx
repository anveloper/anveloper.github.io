## 문제

[BOJ 16170 - 오늘의 날짜는?](https://www.acmicpc.net/problem/16170)

> 현재 UTC 기준의 연도, 월, 일을 출력한다.

## 풀이

C 표준 라이브러리의 `time`과 `gmtime`을 사용하여 현재 UTC 시간을 구한 뒤 연도, 월, 일을 줄바꿈으로 구분하여 출력한다.

## 코드

```cpp
#include <stdio.h>
#include <time.h>
int main()
{
  time_t timer;
  struct tm *t;
  timer = time(NULL);
  t = gmtime(&timer);
  printf("%d\n%d\n%d", t->tm_year + 1900, t->tm_mon + 1, t->tm_mday);
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
