## 문제

[BOJ 2495 - 연속구간](https://www.acmicpc.net/problem/2495)

> 세 개의 8자리 수에 대해 같은 숫자가 연속으로 나타나는 최대 길이를 각각 구한다.

## 풀이

각 문자열을 순회하면서 이전 문자와 같으면 카운트를 증가시키고, 다르면 1로 초기화한다. 매번 최대값을 갱신하여 최종 결과를 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{

  string str;

  for (int i = 0; i < 3; i++)
  {
    cin >> str;
    int cnt = 1;
    int max = 1;
    for (int i = 0; i < str.length() - 1; i++)
    {
      if (str[i] == str[i + 1])
      {
        cnt++;

        if (cnt > max)
        {
          max = cnt;
        }
      }
      else
      {
        cnt = 1;
      }
    }
    cout << max << "\n";
  }
  return 0;
}
```

## 복잡도

- 시간: O(L) (L: 문자열 길이, 고정 8)
- 공간: O(L)
