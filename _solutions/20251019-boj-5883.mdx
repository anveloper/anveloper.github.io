## 문제

[BOJ 5883 - 아이폰 9S](https://www.acmicpc.net/problem/5883)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int n;
bool check[1000010];
vector<int> vec;

int main()
{
  ios_base::sync_with_stdio(false);
  cin.tie(0);
  cout.tie(0);

  cin >> n;
  int temp;
  for (int i = 0; i < n; i++)
  {
    cin >> temp;
    vec.push_back(temp);
    check[temp] = 1;
  }

  int ans = 0;

  for (int i = 0; i < 1000001; i++)
  {
    if (check[i])
    {
      int m = vec[0];
      int m_cnt = 0;
      for (int j = 0; j < n; j++)
      {
        if (vec[j] == i)
          continue;
        if (m == vec[j])
          m_cnt++;
        else if (m != vec[j])
        {
          m = vec[j];
          m_cnt = 1;
        }
        else if (j == n - 1)
        {
          m_cnt++;
        }
        ans = max(ans, m_cnt);
      }
    }
  }

  cout << ans;

  return 0;
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
