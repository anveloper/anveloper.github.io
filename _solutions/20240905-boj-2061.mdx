## 문제

[BOJ 2061 - 좋은 암호](https://www.acmicpc.net/problem/2061)

> 큰 수 K와 정수 L이 주어졌을 때, K가 2 이상 L 미만의 수로 나누어 떨어지면 BAD, 아니면 GOOD을 출력한다.

## 풀이

K가 매우 큰 수이므로 문자열로 입력받고, 나머지 연산을 자릿수별로 수행한다. 2부터 L-1까지 순회하면서 `K mod i`를 각 자릿수에 대해 `(누적값 * 10 + 현재 자릿수) % i`로 계산한다. 나머지가 0인 수를 찾으면 BAD과 함께 해당 수를 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;
int main()
{
  ios::sync_with_stdio(false), cin.tie(NULL);
  string K;
  register int L;
  cin >> K >> L;
  for (register int i = 2, j; i < L; ++i)
  {
    register int ans = 0, tmp = 1;
    for (j = K.length() - 1; j >= 0; --j)
    {
      ans = (ans + (K[j] - '0') * tmp) % i;
      tmp *= 10;
      tmp %= i;
    }
    if (ans == 0)
    {
      cout << "BAD " << i;
      return 0;
    }
  }
  cout << "GOOD";
  return 0;
}
```

## 복잡도

- 시간: O(L * D) (L: 범위 상한, D: K의 자릿수)
- 공간: O(D)
