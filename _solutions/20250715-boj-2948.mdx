## 문제

[BOJ 2948 - 2009년](https://www.acmicpc.net/problem/2948)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
from datetime import datetime


def year_2009(D, M):
    weekday_list = [
        "Monday",
        "Tuesday",
        "Wednesday",
        "Thursday",
        "Friday",
        "Saturday",
        "Sunday",
    ]

    return weekday_list[datetime(year=2009, month=M, day=D).weekday()]


if __name__ == "__main__":
    D, M = map(int, input().split())
    print(year_2009(D, M))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
