## 문제

[BOJ 3213 - 피자](https://www.acmicpc.net/problem/3213)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```python
from math import ceil
import sys

input = sys.stdin.readline

N = int(input())
slice1 = 0
slice2 = 0
slice3 = 0
ans = 0

for i in range(N):
    x = input().rstrip()
    if x == "1/4":
        slice1 += 1
    if x == "3/4":
        slice2 += 1
    if x == "1/2":
        slice3 += 1

if slice1 <= slice2:
    ans = slice2 + ceil(slice3 / 2)
else:
    ans = slice2 + ceil(slice3 / 2) + ceil((slice1 - slice2 - (slice3 % 2 * 2)) / 4)
print(ans)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
