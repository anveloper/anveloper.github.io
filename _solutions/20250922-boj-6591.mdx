## 문제

[BOJ 6591 - 이항 쇼다운](https://www.acmicpc.net/problem/6591)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;

void swap(int *a, int *b)
{
  int *temp = a;
  a = b;
  b = temp;
}

int main()
{
  while (true)
  {
    int n, k;
    cin >> n >> k;
    if (!n && !k)
      break;
    int r = n - k;
    if (r > k)
      swap(r, k);

    int div = 1;
    long long ans = 1;
    for (int i = n; i >= n - r + 1; i--)
    {
      ans *= i;
      ans /= div;
      div++;
    }

    cout << ans << '\n';
  }
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
