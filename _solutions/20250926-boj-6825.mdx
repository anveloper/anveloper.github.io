## 문제

[BOJ 6825 - Body Mass Index](https://www.acmicpc.net/problem/6825)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>

int main()
{
  float a, b;
  std::cin >> a >> b;

  float bmi{a / (b * b)};

  if (bmi < 18.5)
  {
    std::cout << "Underweight";
  }
  else if (bmi <= 25)
  {
    std::cout << "Normal weight";
  }
  else
  {
    std::cout << "Overweight";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
