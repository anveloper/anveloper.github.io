## 문제

[BOJ 1398 - 동전 문제](https://www.acmicpc.net/problem/1398)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```python
import sys
input = sys.stdin.readline

dp = [0] * 101
for i in range(1, 101):
    dp[i] = dp[i - 1] + 1
    if i >= 10:
        dp[i] = min(dp[i], dp[i - 10] + 1)

    if i >= 25:
        dp[i] = min(dp[i], dp[i - 25] + 1)

for _ in range(int(input())):
    n = int(input())
    result = 0
    while n:
        result += dp[n % 100]
        n //= 100

    print(result)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
