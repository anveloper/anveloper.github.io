## 문제

[BOJ 1837 - 암호제작](https://www.acmicpc.net/problem/1837)

> 두 소수의 곱 P가 주어질 때, K보다 작은 소인수가 존재하면 BAD, 아니면 GOOD을 출력한다.

## 풀이

2부터 K-1까지 순회하며 P를 나눌 수 있는 수가 있으면 BAD과 해당 수를 출력한다. 없으면 GOOD을 출력한다.

## 코드

```python
P, K = map(int, input().split())

for i in range(2, K):
  if P % i == 0:
    print("BAD", i)
    break
else:
  print("GOOD")
```

## 복잡도

- 시간: O(K)
- 공간: O(1)
