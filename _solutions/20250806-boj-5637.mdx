## 문제

[BOJ 5637 - 가장 긴 단어](https://www.acmicpc.net/problem/5637)

## 풀이

문자열 처리를 통해 문제를 해결한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

string preprocess(string str)
{
  string result = "";

  for (auto iter = str.begin(); iter != str.end(); iter++)
  {
    if (isalpha(*iter) || *iter == '-')
    {
      result += *iter;
    }
  }
  return result;
}

int main()
{
  ios_base::sync_with_stdio(false);
  cin.tie(nullptr);

  string max = "";
  string str;

  while (true)
  {
    cin >> str;
    if (str == "E-N-D")
      break;
    str = preprocess(str);
    max = max.length() < str.length() ? str : max;
  }

  for (auto iter = max.begin(); iter != max.end(); iter++)
    *iter = tolower(*iter);

  cout << max;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
