## 문제

[BOJ 3982 - Soccer Bets](https://www.acmicpc.net/problem/3982)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <unordered_map>
#include <array>
#include <string>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int c;
  if (!(cin >> c))
    return 0;

  while (c--)
  {
    unordered_map<string, array<int, 2>> stat; // [0]=wins, [1]=losses

    for (int i = 0; i < 16; ++i)
    {
      string t1, t2;
      int g1, g2;
      cin >> t1 >> t2 >> g1 >> g2;

      if (g1 > g2)
      {
        ++stat[t1][0]; // t1 win
        ++stat[t2][1]; // t2 loss
      }
      else
      {
        ++stat[t2][0]; // t2 win
        ++stat[t1][1]; // t1 loss
      }
    }

    string champion;
    for (const auto &kv : stat)
    {
      if (kv.second[0] == 4 && kv.second[1] == 0)
      {
        champion = kv.first;
        break; // 유일하므로 즉시 종료
      }
    }

    cout << champion << '\n';
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
