## 문제

[BOJ 1434 - 책 정리](https://www.acmicpc.net/problem/1434)

> N개의 상자에 M권의 책을 넣은 후, 상자들에 남는 공간의 합을 구한다.

## 풀이

각 책을 들어갈 수 있는 첫 번째 상자에 넣고, 남은 공간을 합산한다.

## 코드

```javascript
const solution = (input) => {
  const [N, M] = input[0].split(" ").map(Number);
  const [boxes, books] = [input[1].split(" ").map(Number), input[2].split(" ").map(Number)];
  for (let j = 0; j < M; j++) {
    const book = books[j];
    for (let i = 0; i < N; i++) { if (book <= boxes[i]) { boxes[i] -= book; break; } }
  }
  return boxes.reduce((acc, cur) => (acc += cur), 0);
};
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N x M)
- 공간: O(N)
