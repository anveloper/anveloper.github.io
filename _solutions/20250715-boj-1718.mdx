## 문제

[BOJ 1718 - 암호](https://www.acmicpc.net/problem/1718)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
text, key = input(), input()

answer = ''

for i in range(len(text)):
    if text[i] == ' ': answer += ' '
    else: answer += chr((ord(text[i]) - ord(key[i%len(key)]) - 1) % 26 + ord('a'))

print(answer)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
