## 문제

[BOJ 15964 - 이상한 기호](https://www.acmicpc.net/problem/15964)

> 두 정수 A, B가 주어질 때 (A+B) * (A-B)의 값을 구한다.

## 풀이

두 수를 입력받아 (A+B) * (A-B)를 계산하여 출력한다. 오버플로 방지를 위해 `long long` 타입을 사용한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(int argc, char *argv[])
{
  long long a, b;
  cin >> a >> b;
  cout << (a + b) * (a - b) << endl;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
