## 문제

[BOJ 12840 - 창용이의 시계](https://www.acmicpc.net/problem/12840)

> 초기 시각이 주어지고, 시간을 앞으로/뒤로 이동하거나 현재 시각을 출력하는 쿼리를 처리한다.

## 풀이

시각을 전체 초 단위로 변환하여 관리한다. 쿼리 1은 초를 더하고, 쿼리 2는 초를 빼며, 쿼리 3은 현재 시각을 시/분/초로 변환하여 출력한다. 24시간(86400초) 범위를 벗어나지 않도록 모듈러 연산을 사용한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  cin.tie(0)->sync_with_stdio(0);

  int h, m, s, t, n, x, y;
  cin >> h >> m >> s >> n;
  t = 3600 * h + 60 * m + s;
  for (int i = 0; i < n; i++)
  {
    cin >> x;

    if (x == 1)
      cin >> y, t = (t + y) % (24 * 60 * 60);
    else if (x == 2)
    {
      cin >> y;
      t = (t - y) % (24 * 60 * 60);
      while (t < 0)
        t += (24 * 60 * 60);
    }
    else if (x == 3)
      cout << t / 3600 << ' ' << (t / 60) % 60 << ' ' << t % 60 << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
