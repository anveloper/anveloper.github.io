## 문제

[BOJ 2986 - 파스칼](https://www.acmicpc.net/problem/2986)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys

input = sys.stdin.readline

N = int(input())

i = 2
Counter = 1
while i * i <= N:
    if N % i == 0:
        Counter = N // i
        break
    i += 1
print(N - Counter)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
