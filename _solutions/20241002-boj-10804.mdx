## 문제

[BOJ 10804 - 카드 역배치](https://www.acmicpc.net/problem/10804)

> 1부터 20까지의 카드가 순서대로 놓여있을 때, 주어진 구간을 10번 뒤집은 후 최종 카드 배열을 출력한다.

## 풀이

배열을 1~20으로 초기화한 뒤, 10번의 구간 뒤집기를 수행한다. 각 뒤집기는 구간의 양 끝에서 swap하며 안쪽으로 좁혀나간다.

## 코드

```cpp
#include <iostream>
using namespace std;
int arr[21];
int main()
{
  ios::sync_with_stdio(0);
  cin.tie(0);

  for (int i = 1; i < 21; i++)
  {
    arr[i] = i;
  }

  for (int i = 0; i < 10; i++)
  {
    int s, e, a, b;
    cin >> s >> e;
    a = s;
    b = e;
    for (int j = 0; j < (e - s + 1) / 2; j++)
    {
      swap(arr[a++], arr[b--]);
    }
  }

  for (int i = 1; i < 21; i++)
  {
    cout << arr[i] << " ";
  }
  return 0;
}
```

## 복잡도

- 시간: O(1) (최대 10 * 20 = 고정 크기)
- 공간: O(1)
