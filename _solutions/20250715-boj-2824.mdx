## 문제

[BOJ 2824 - 최대공약수](https://www.acmicpc.net/problem/2824)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline


def gcd(a, b):
    while b > 0:
        tmp = a % b
        a = b
        b = tmp
    return a


def multiply(lst):
    return eval("*".join([str(n) for n in lst]))


N = int(input())
N_lst = list(map(int, input().split()))

M = int(input())
M_lst = list(map(int, input().split()))

a = multiply(N_lst)
b = multiply(M_lst)

print("%s" % str(gcd(a, b))[-9:])
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
