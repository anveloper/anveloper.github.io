## 문제

[BOJ 1590 - 캠프가는 영식](https://www.acmicpc.net/problem/1590)

> N개의 버스 노선이 있을 때, 시각 T 이후 가장 빨리 탈 수 있는 버스까지의 대기 시간을 구한다.

## 풀이

각 버스 노선의 출발 시간 목록을 생성한 뒤, 마지막 출발 시간이 T보다 작으면 건너뛴다. 그렇지 않으면 이분 탐색으로 T 이상인 가장 빠른 출발 시간을 찾아 대기 시간을 계산하고, 모든 노선 중 최솟값을 출력한다.

## 코드

```python
N, T = map(int, input().split())
result = []
for _ in range(N):
    S, I, C = map(int, input().split())
    time = [S + (I * c) for c in range(C)]
    if time[-1] < T:
        continue

    start = 0
    end = C - 1
    answer = 0
    while start <= end:
        mid = (start + end) // 2
        if time[mid] >= T:
            answer = mid
            end = mid - 1
        else:
            start = mid + 1

    result.append(time[answer] - T)

if result:
    print(min(result))
else:
    print(-1)
```

## 복잡도

- 시간: O(N * log C)
- 공간: O(C)
