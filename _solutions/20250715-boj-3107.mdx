## 문제

[BOJ 3107 - IPv6](https://www.acmicpc.net/problem/3107)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
ipv6 = input()
ipv6 = ipv6.split(":")
if ipv6[0] == "":
    ipv6 = ipv6[1:]
if ipv6[-1] == "":
    ipv6 = ipv6[:-1]

result = ""
for i in ipv6:
    if i == "":
        result += "0000:" * (8 - len(ipv6) + 1)
    else:
        result += i.zfill(4) + ":"

print(result[:-1])
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
