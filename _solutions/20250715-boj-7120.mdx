## 문제

[BOJ 7120 - String](https://www.acmicpc.net/problem/7120)

## 풀이

문자열 처리를 통해 문제를 해결한다.

## 코드

```python
a = input()
b = ""
for i in a:
    if b != i:
        print(i, end="")
    b = i
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
