## 문제

[BOJ 1503 - 세 수 고르기](https://www.acmicpc.net/problem/1503)

> 집합 S에 포함되지 않는 세 자연수를 골라 그 곱과 N의 차이의 절댓값을 최소화한다.

## 풀이

1부터 1001까지의 수 중 S에 포함되지 않는 수를 세 개 골라 곱을 구한다. 세 번째 수의 곱이 N+1을 초과하면 더 이상 탐색할 필요가 없으므로 내부 루프를 중단하는 가지치기를 적용한다. 모든 조합 중 |N - i*j*k|의 최솟값을 출력한다.

## 코드

```python
import sys

input = sys.stdin.readline
INF = sys.maxsize

n, m = list(map(int, input().rstrip().split()))
s = list(map(int, input().rstrip().split()))
res = INF

for i in range(1, 1002):
    if i in s:
        continue
    for j in range(1, 1002):
        if j in s:
            continue
        for k in range(1, 1002):
            if k in s:
                continue
            q = i * j * k
            if abs(n - q) < res:
                res = abs(n - q)
            if n + 1 < q:
                break

print(res)
```

## 복잡도

- 시간: O(1001^3) (가지치기로 실제 수행 시간은 크게 감소)
- 공간: O(M) (M은 집합 S의 크기)
