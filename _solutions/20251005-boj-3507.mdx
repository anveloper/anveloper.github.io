## 문제

[BOJ 3507 - Automated Telephone Exchange](https://www.acmicpc.net/problem/3507)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int n;
int main()
{
  cin >> n;
  if (n > 198)
    cout << 0;
  else
    cout << 99 - (n - 99) + 1;
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
