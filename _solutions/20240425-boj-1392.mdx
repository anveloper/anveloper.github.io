## 문제

[BOJ 1392 - 노래 악보](https://www.acmicpc.net/problem/1392)

> 각 노래의 재생 시간이 주어질 때, 특정 시각에 어떤 노래가 재생 중인지 출력한다.

## 풀이

각 노래의 시간만큼 배열을 확장하여 시간 -> 노래 번호 매핑 테이블을 만든다.

## 코드

```javascript
const solution = (input) => {
  const [N, Q] = input[0].split(" ").map((v) => +v);
  const arr = [];
  for (let i = 1; i < N + 1; i++) {
    time = +input[i];
    for (let j = 0; j < time; j++) arr.push(i);
  }
  const sb = [];
  for (let i = N + 1; i < N + 1 + Q; i++) sb.push(arr[input[i]]);
  return sb.join("\n");
};
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N x T + Q)
- 공간: O(N x T)
