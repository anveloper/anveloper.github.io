## 문제

[BOJ 8718 - Bałwanek](https://www.acmicpc.net/problem/8718)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

#define ll long long
using namespace std;

ll x, k, a, b, c;
int main()
{
  cin >> x >> k;
  if (7000 * k <= 1000 * x)
    a = 7000 * k;
  if (3500 * k <= 1000 * x)
    b = 3500 * k;
  if (1750 * k <= 1000 * x)
    c = 1750 * k;
  cout << max({a, b, c});
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
