## 문제

[BOJ 1497 - 기타콘서트](https://www.acmicpc.net/problem/1497)

> N개의 기타 중 일부를 선택하여 최대한 많은 곡을 연주할 수 있되, 기타 수를 최소화한다.

## 풀이

각 기타가 연주할 수 있는 곡을 비트마스크로 표현한 뒤, 모든 기타 부분집합(2^N)을 순회하며 OR 연산으로 연주 가능한 곡 수를 구한다. 최대 곡 수를 달성하는 부분집합 중 기타 수가 최소인 것을 선택한다.

## 코드

```python
import sys

n, m = map(int, input().split(" "))
min_g = n + 1
max_s = 0
l = []
for i in range(n):
    guiter, able = sys.stdin.readline().rstrip().split(" ")
    able = able.replace("Y", "1")
    able = able.replace("N", "0")
    l.append(int(able, 2))


for i in range(1 << n):
    n_guitar = 0
    check = 0
    for j in range(n):
        if i & (1 << j):
            a = l[j]
            check = check | a
            n_guitar += 1
    if max_s < bin(check).count("1"):
        min_g = n_guitar
        max_s = bin(check).count("1")

    elif max_s == bin(check).count("1"):
        if min_g > n_guitar:
            min_g = n_guitar

if max_s == 0:
    print(-1)
else:
    print(min_g)
```

## 복잡도

- 시간: O(2^N * N)
- 공간: O(N)
