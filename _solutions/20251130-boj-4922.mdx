## 문제

[BOJ 4922 - Walk Like an Egyptian](https://www.acmicpc.net/problem/4922)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  while (1)
  {
    int n, sum = 0, cnt = 1;
    cin >> n;
    if (!n)
      break;
    for (int i = 0; i < n; i++, cnt += 2)
      sum += cnt;
    cout << n << " => " << sum - n + 1 << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
