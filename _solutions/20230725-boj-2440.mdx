## 문제

[BOJ 2440 - 별 찍기 - 3](https://www.acmicpc.net/problem/2440)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day549;

import java.util.*;

public class Day534BOJ2440별찍기3 {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);

    int num = sc.nextInt();

    for (int i = 0; i < num; i++) {

      for (int j = i + 1; j <= num; j++) {
        System.out.print("*");
      }
      System.out.println();
    }
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
