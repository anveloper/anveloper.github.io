## 문제

[BOJ 15953 - 상금 헌터](https://www.acmicpc.net/problem/15953)

> 두 대회에서 각각의 등수가 주어질 때, 받을 수 있는 총 상금을 구한다.

## 풀이

각 대회의 등수별 상금 테이블을 미리 정의한다. 첫 번째 대회는 1등부터 순서대로 1명, 2명, 3명, ... 인원이 배정되고, 두 번째 대회는 2의 거듭제곱(1, 2, 4, 8, 16명)으로 배정된다. 등수에서 해당 구간의 인원을 차감하며 어느 구간에 속하는지 판별하여 상금을 합산한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  cin.tie(0)->sync_with_stdio(0);
  int prize1[6] = {500, 300, 200, 50, 30, 10};
  int prize2[5] = {512, 256, 128, 64, 32};
  int t;
  cin >> t;
  while (t--)
  {
    int a, b, sum = 0;
    cin >> a >> b;
    if (a)
    {
      for (int i = 0; i < 6; i++)
      {
        a -= i + 1;
        if (a <= 0)
        {
          sum += prize1[i];
          break;
        };
      }
    }
    if (b)
    {
      for (int i = 0; i < 5; i++)
      {
        b -= (1 << i);
        if (b <= 0)
        {
          sum += prize2[i];
          break;
        }
      }
    }
    cout << sum * 10000 << '\n';
  }
}
```

## 복잡도

- 시간: O(T) (T: 테스트 케이스 수, 내부 반복은 상수)
- 공간: O(1)
