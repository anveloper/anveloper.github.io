## 문제

[BOJ 2755 - 이번학기 평점은 몇점?](https://www.acmicpc.net/problem/2755)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
d = {"A+": 4.3, "A0": 4.0, "A-": 3.7, "B+": 3.3, "B0": 3.0, 
     "B-": 2.7, "C+": 2.3, "C0": 2.0, "C-": 1.7, "D+": 1.3, 
     "D0": 1.0, "D-": 0.7, "F": 0.0}
N = t = 0
for _ in range(int(input())):
    s, n, g = input().split()
    n = int(n) 
    N += n
    t += n*d[g]
print("%.2f" % (round(t/N + 10**-10, 2)))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
