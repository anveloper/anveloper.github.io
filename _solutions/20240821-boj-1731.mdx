## 문제

[BOJ 1731 - 추론](https://www.acmicpc.net/problem/1731)

> 주어진 수열이 등차수열인지 등비수열인지 판별하고, 다음 항을 출력한다.

## 풀이

처음 세 항의 차이를 비교하여 `p[1] - p[0]`과 `p[2] - p[1]`이 같으면 등차수열로 판단하고 마지막 항에 공차를 더한다. 그렇지 않으면 등비수열로 판단하고 마지막 항에 공비를 곱한다.

## 코드

```cpp
#include <stdio.h>

main()
{
  int n, p[50];
  scanf("%d", &n);
  for (int i = 0; i < n; i++)
    scanf("%d", &p[i]);
  if (p[1] - p[0] == p[2] - p[1])
    printf("%d", p[n - 1] + (p[1] - p[0]));
  else
    printf("%d", p[n - 1] * (p[1] / p[0]));
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
