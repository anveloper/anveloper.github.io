## 문제

[BOJ 6768 - Don’t pass me the ball!](https://www.acmicpc.net/problem/6768)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int n, ans = 1;
int main()
{
  cin >> n;
  for (int i = 1; i <= 3; i++)
    ans *= n - i;
  cout << ans / 6;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
