## 문제

[BOJ 2965 - 캥거루 세마리](https://www.acmicpc.net/problem/2965)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
A, B, C = map(int, input().split())
print(max(B - A - 1, C - B - 1))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
