## 문제

[BOJ 2558 - A+B - 2](https://www.acmicpc.net/problem/2558)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day549;

import java.util.Scanner;

public class Day519BOJ2558ab2 {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int a = sc.nextInt();
    int b = sc.nextInt();
    sc.close();

    System.out.println(a + b);
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
