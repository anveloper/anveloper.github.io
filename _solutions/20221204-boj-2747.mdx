## 문제

[BOJ 2747 - 피보나치 수](https://www.acmicpc.net/problem/2747)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```java
package day349;

import java.util.Scanner;

public class Day300BOJ2747피보나치 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int F1 = 0, F2 = 1, F3 = 1;
        for (int i = 2; i <= N; i++) {
            F3 = F1 + F2;
            F1 = F2;
            F2 = F3;
        }
        System.out.println(F3);
        sc.close();
    }
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
