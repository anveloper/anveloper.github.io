## 문제

[BOJ 8246 - Stół](https://www.acmicpc.net/problem/8246)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;
int a, b, k;
int main()
{
  cin >> a >> b >> k;
  cout << (a / k) * (b / k) - max(0, a / k - 2) * max(0, b / k - 2);
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
