## 문제

[BOJ 9085 - 더하기](https://www.acmicpc.net/problem/9085)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  int c, n, num, sum = 0;
  cin >> c;
  for (int i = 0; i < c; i++)
  {
    cin >> n;
    for (int j = 0; j < n; j++)
    {
      cin >> num;
      sum += num;
    }
    cout << sum << '\n';
    sum = 0;
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
