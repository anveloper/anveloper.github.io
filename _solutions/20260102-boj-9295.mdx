## 문제

[BOJ 9295 - 주사위](https://www.acmicpc.net/problem/9295)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int test, fNum, lNum;
  cin >> test;
  int *sum = new int[test];

  for (int i = 0; i < test; i++)
  {
    cin >> fNum >> lNum;
    sum[i] = fNum + lNum;
  }
  for (int i = 1; i <= test; i++)
  {
    cout << "Case " << i << ": " << sum[i - 1] << '\n';
  }
  delete[] sum;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
