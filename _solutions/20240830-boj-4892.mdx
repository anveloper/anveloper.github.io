## 문제

[BOJ 4892 - 숫자 맞추기 게임](https://www.acmicpc.net/problem/4892)

> 입력된 수에 3을 곱한 후 홀짝에 따라 다른 연산을 수행하여 최종 결과를 출력한다.

## 풀이

N에 3을 곱한 뒤, 홀수이면 `(3N+1)/2`를 다시 3배하고 9로 나누며, 짝수이면 `3N/2`를 3배하고 9로 나눈다. 홀짝 여부와 최종 결과를 순서 번호와 함께 출력한다. 0이 입력되면 종료한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int n, i = 1;

  while (i)
  {
    cin >> n;
    if (n == 0)
      break;
    cout << i << ". ";
    n *= 3;
    if (n % 2)
    {
      cout << "odd ";
      n = (n + 1) / 2;
      n = 3 * n;
      n = n / 9;
      cout << n << "\n";
    }
    else
    {
      cout << "even ";
      n = n / 2;
      n = 3 * n;
      n = n / 9;
      cout << n << "\n";
    }
    i++;
  }

  return 0;
}
```

## 복잡도

- 시간: O(T) (T: 테스트 케이스 수)
- 공간: O(1)
