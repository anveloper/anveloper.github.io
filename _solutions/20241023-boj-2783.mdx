## 문제

[BOJ 2783 - 삼각 김밥](https://www.acmicpc.net/problem/2783)

> 여러 삼각 김밥의 가격과 무게가 주어질 때, 1000g당 가장 저렴한 가격을 구한다.

## 풀이

첫 번째 삼각 김밥의 g당 가격을 기준으로 시작하여, 이후 입력되는 각 삼각 김밥과 비교해 더 저렴한 g당 가격이 있으면 갱신한다. 최종 최솟값에 1000을 곱하여 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  double S_25;
  int a, b;
  cin >> a >> b;
  double min = ((double)a / b);

  int n;
  cin >> n;

  for (int i = 0; i < n; i++)
  {
    cin >> a >> b;
    min = min > ((double)a / b) ? ((double)a / b) : min;
  }

  cout << min * 1000 << "\n";
}
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
