## 문제

[BOJ 4436 - 엘프의 검](https://www.acmicpc.net/problem/4436)

> N의 배수들을 순서대로 확인하면서, 지금까지 나온 배수들의 자릿수에 0부터 9까지 모든 숫자가 등장하는 최소 배수 i를 구한다.

## 풀이

N × 1, N × 2, ... 순서대로 각 배수의 자릿수를 집합에 추가한다. 집합의 크기가 10이 되면(0~9 모두 등장) 해당 배수 번호 i를 출력하고 종료한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main()
{
  cin.tie(0)->sync_with_stdio(0);
  ll n;
  while (cin >> n)
  {
    set<int> s;
    for (int i = 1; i < 1000; i++)
    {
      for (char c : to_string(n * i))
        s.insert(c);
      if (s.size() == 10)
      {
        cout << i << "\n";
        break;
      }
    }
  }
  return 0;
}
```

## 복잡도

- 시간: O(I × D) (I: 최소 배수 번호, D: 자릿수)
- 공간: O(1)
