## 문제

[BOJ 4806 - 줄 세기](https://www.acmicpc.net/problem/4806)

> 입력이 끝날 때까지 총 몇 줄이 입력되었는지 세어 출력한다.

## 풀이

`getline`으로 한 줄씩 읽으며 EOF까지 카운트를 증가시킨 뒤 총 줄 수를 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

string s;
int ans;

int main()
{
  while (getline(cin, s))
    ans++;
  cout << ans;
}
```

## 복잡도

- 시간: O(N) (N: 줄 수)
- 공간: O(L) (L: 한 줄의 최대 길이)
