## 문제

[BOJ 4107 - Knitting](https://www.acmicpc.net/problem/4107)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
	ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
	while (1) {
		int i, n, m, k, p[100], ans; cin >> n >> m >> k;
		if (n == 0)break;
		ans = n;
		for (i = 0; i < k; i++) cin >> p[i];
		for (i = 0; i < m-1; i++) {
			n += p[i % k];
			ans += n;
		}
		cout << ans << '\n';
	}
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
