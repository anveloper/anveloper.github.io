## 문제

[BOJ 1980 - 햄버거 사랑](https://www.acmicpc.net/problem/1980)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main(void)
{
  int n, m, t;
  cin >> n >> m >> t;
  int mx = max(n, m), mn = min(n, m);

  if (t < mn)
  {
    cout << 0 << ' ' << t << '\n';
  }
  for (int i = t; i > 0; i--)
  {
    int coke = t - i;
    for (int j = 0; j <= (i / mx); j++)
    {
      int tmp = i - (j * mx);
      if (tmp % mn == 0)
      {
        int bg = (tmp / mn) + j;
        cout << bg << ' ' << coke << '\n';
        return 0;
      }
    }
  }
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
