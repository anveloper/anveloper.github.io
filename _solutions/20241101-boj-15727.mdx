## 문제

[BOJ 15727 - 조별과제를 하려는데 조장이 사라졌다](https://www.acmicpc.net/problem/15727)

> 거리 N이 주어졌을 때, 5km/h로 이동하는 데 걸리는 시간(올림)을 구한다.

## 풀이

(N-1)/5 + 1로 올림 나눗셈을 계산한다. N을 5로 나눈 몫에서 나머지가 있으면 1을 추가하는 것과 동일하다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(int argc, char *argv[])
{
  int n;
  cin >> n;
  cout << (n - 1) / 5 + 1 << endl;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
