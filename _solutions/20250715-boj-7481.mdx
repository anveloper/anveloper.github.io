## 문제

[BOJ 7481 - ATM놀이](https://www.acmicpc.net/problem/7481)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
T = int(input())
for i in range(T):
    a, b, S = map(int, input().split())
    sw = 0
    if b < a:
        sw = 1
        b, a = a, b

    ans = 0
    t = S // b * b
    for j in range(a + 2):
        if (S - t) % a == 0:
            ans = [(S - t) // a, t // b]
            break
        else:
            t -= b

    if ans == 0 or ans[0] < 0 or ans[1] < 0:
        print("Impossible")
    else:
        if sw == 1:
            ans.reverse()
        print(*ans)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
