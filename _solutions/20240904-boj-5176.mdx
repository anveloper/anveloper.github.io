## 문제

[BOJ 5176 - 대회 자리](https://www.acmicpc.net/problem/5176)

> P명의 참가자가 M개의 자리 중 하나를 선택할 때, 이미 선택된 자리를 고른 참가자 수를 구한다.

## 풀이

크기 501의 불리언 배열로 자리 배정 여부를 관리한다. 각 참가자가 선택한 자리가 이미 배정되어 있으면 충돌 카운트를 증가시키고, 아니면 해당 자리를 배정 완료로 표시한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int k = 0;
  cin >> k;
  while (k--)
  {
    int p = 0, m = 0;
    cin >> p >> m;
    bool arr[501] = {};
    int cnt = 0;
    for (int i = 0; i < p; i++)
    {
      int now = 0;
      cin >> now;
      if (arr[now])
        cnt++;
      else
        arr[now] = 1;
    }
    cout << cnt << endl;
  }
}
```

## 복잡도

- 시간: O(K * P) (K: 테스트 케이스 수, P: 참가자 수)
- 공간: O(M) (M: 자리 수)
