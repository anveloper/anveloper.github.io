## 문제

[BOJ 2545 - 팬케익 먹기](https://www.acmicpc.net/problem/2545)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;
typedef long long ll;

int t;
ll a[5], d;

int main()
{
  scanf("%d", &t);
  while (t--)
  {
    scanf("%lld %lld %lld %lld", &a[0], &a[1], &a[2], &d);
    sort(a, a + 3);
    ll tmp = a[0] + a[1] + a[2] - d;
    if (a[0] >= tmp / 3)
      a[0] = tmp / 3;
    tmp -= a[0];
    if (a[1] >= tmp / 2)
      a[1] = tmp / 2;
    tmp -= a[1];
    a[2] = tmp;
    printf("%lld\n", a[0] * a[1] * a[2]);
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
