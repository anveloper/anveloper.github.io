## 문제

[BOJ 13163 - 닉네임에 갓 붙이기](https://www.acmicpc.net/problem/13163)

> 공백으로 구분된 닉네임에서 첫 번째 단어를 "god"으로 교체하고 나머지 단어를 이어 붙여 출력한다.

## 풀이

한 줄씩 입력받아 공백을 기준으로 단어를 분리한다. 첫 번째 단어는 "god"으로 대체하고, 이후 단어들은 공백 없이 이어 붙여 결과를 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
  int n;
  cin >> n;
  cin.ignore();

  while (n--)
  {
    string s;
    string ans = "";
    int cnt = 0;
    cin.clear();
    getline(cin, s);

    for (int i = 0; i < s.size(); i++)
    {

      string tmp = "";
      int j = i;
      while (s[j] != ' ' && j != s.size())
        tmp += s[j++];
      i = j;
      if (!cnt)
      {
        ans += "god";
        cnt = 1;
        continue;
      }
      else
      {
        ans += tmp;
      }
    }
    cout << ans << '\n';
  }
}
```

## 복잡도

- 시간: O(N × L) (L: 각 줄의 길이)
- 공간: O(L)
