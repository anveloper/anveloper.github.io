## 문제

[BOJ 2966 - 찍기](https://www.acmicpc.net/problem/2966)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
N = int(input())
S = input()
Adrian = "ABC" * (N // 3) + "ABC"[: N % 3]
Bruno = "BABC" * (N // 4) + "BABC"[: N % 4]
Goran = "CCAABB" * (N // 6) + "CCAABB"[: N % 6]

A = 0
B = 0
G = 0
for i in range(N):
    if S[i] == Adrian[i]:
        A += 1
    if S[i] == Bruno[i]:
        B += 1
    if S[i] == Goran[i]:
        G += 1

print(max(A, B, G))
if max(A, B, G) == A:
    print("Adrian")
if max(A, B, G) == B:
    print("Bruno")
if max(A, B, G) == G:
    print("Goran")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
