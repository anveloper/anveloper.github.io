## 문제

[BOJ 2475 - 검증수](https://www.acmicpc.net/problem/2475)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day599;

import java.util.*;

public class Day571BOJ2475검증수 {
  public static void main(String[] args) throws Exception {
    Scanner sc = new Scanner(System.in);

    int sum = 0;
    for (int i = 0; i < 5; i++) {
      int tmp = sc.nextInt();
      sum += tmp * tmp;
    }
    int ans = sum % 10;
    System.out.println(ans);
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
