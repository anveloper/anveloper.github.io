## 문제

[BOJ 5341 - Pyramids](https://www.acmicpc.net/problem/5341)

> 피라미드의 맨 아래 줄 블록 수 N이 주어지면, 전체 블록 수(1+2+...+N)를 구한다. 0이 입력되면 종료한다.

## 풀이

N이 0이 아닐 때까지 반복하며, 1부터 N까지의 합을 직접 반복문으로 계산하여 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int n;
  while (1)
  {
    cin >> n;
    if (!n)
      break;
    int sum = 0;
    for (int i = 1; i <= n; i++)
      sum += i;
    cout << sum << "\n";
  }
}
```

## 복잡도

- 시간: O(T * N)
- 공간: O(1)
