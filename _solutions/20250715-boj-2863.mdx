## 문제

[BOJ 2863 - 이게 분수?](https://www.acmicpc.net/problem/2863)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
a, b = map(int, input().split())
c, d = map(int, input().split())
f = []
f.append(a / c + b / d)
f.append(c / d + a / b)
f.append(d / b + c / a)
f.append(b / a + d / c)
f_max = max(f)
print(f.index(f_max))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
