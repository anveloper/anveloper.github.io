## 문제

[BOJ 2557 - Hello World](https://www.acmicpc.net/problem/2557)

> "Hello World!"를 출력한다.

## 풀이

printf로 "Hello World!" 문자열을 그대로 출력한다.

## 코드

```cpp
#include <stdio.h>

int main()
{
  printf("Hello World!");
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
