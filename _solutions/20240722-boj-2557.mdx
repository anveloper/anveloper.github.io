## 문제

[BOJ 2557 - Hello World](https://www.acmicpc.net/problem/2557)

"Hello World!"를 출력하라.

### 입력

없음.

### 출력

"Hello World!"를 출력한다.

### 예제

| 입력 | 출력 |
| --- | --- |
| (없음) | `Hello World!` |

## 풀이

printf로 고정 문자열을 출력하는 기초 입출력 문제이다.

1. `printf("Hello World!")`로 문자열을 출력한다

**핵심 아이디어**: 프로그래밍 언어의 기본 출력 함수 사용법을 익히는 문제이다.

## 코드

```cpp
#include <stdio.h>

int main()
{
  printf("Hello World!");
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
