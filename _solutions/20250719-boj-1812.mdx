## 문제

[BOJ 1812 - 사탕](https://www.acmicpc.net/problem/1812)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main(void)
{
  int n;
  int arr[1000], sum = 0;
  cin >> n;
  for (int i = 0; i < n; i++)
  {
    cin >> arr[i];
    sum += arr[i];
  }
  sum /= 2;
  for (int i = 0; i < n; i++)
  {
    int tmp = 0;
    for (int j = 0; j < n; j += 2)
      tmp += arr[(i + j) % n];
    cout << tmp - sum << '\n';
  }
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
