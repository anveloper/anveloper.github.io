## 문제

[BOJ 2246 - 콘도 선정](https://www.acmicpc.net/problem/2246)

> 거리와 비용이 주어진 콘도 중, 자신보다 가깝고 저렴한 콘도가 없는 후보의 수를 구한다.

## 풀이

콘도를 거리순으로 정렬한 뒤, 순서대로 순회하며 지금까지의 최소 비용보다 현재 비용이 더 작은 경우만 후보로 카운트한다. 거리가 가까운 것부터 보므로, 이전까지 최소 비용보다 저렴해야 더 가까운 곳에서 더 저렴한 곳이 없다는 조건을 만족한다.

## 코드

```python
import sys

input = sys.stdin.readline
A = sorted(tuple(map(int, input().split())) for _ in range(int(input())))
m = 20000
ans = 0
for _, y in A:
    if y < m:
        ans += 1
        m = y
print(ans)
```

## 복잡도

- 시간: O(N * log N)
- 공간: O(N)
