## 문제

[BOJ 4312 - 3의 제곱](https://www.acmicpc.net/problem/4312)

## 풀이

비트마스킹을 활용하여 상태를 관리한다.

## 코드

```python
while True:
    n = int(input())
    if n == 0:
        break
    n -= 1
    print("{", end=" ")
    i = 0
    while n != 0:
        if n % 2:
            print(3**i, end=", " if n // 2 else " ")
        n //= 2
        i += 1
    print("}")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
