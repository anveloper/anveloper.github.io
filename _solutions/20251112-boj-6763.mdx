## 문제

[BOJ 6763 - Speed fines are not fine!](https://www.acmicpc.net/problem/6763)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int a, b, diff;
int main()
{
  cin >> a >> b;
  diff = a - b;
  if (diff >= 0)
    cout << "Congratulations, you are within the speed limit!";
  else
  {
    cout << "You are speeding and your fine is $";
    if (1 <= -diff && -diff <= 20)
      cout << 100;
    else if (21 <= -diff && -diff <= 30)
      cout << 270;
    else if (-diff >= 31)
      cout << 500;
    cout << ".";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
