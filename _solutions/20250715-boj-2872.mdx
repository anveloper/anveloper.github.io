## 문제

[BOJ 2872 - 우리집엔 도서관이 있어](https://www.acmicpc.net/problem/2872)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```python
n, *r = map(int, open(0).read().split())
while r:
    n -= r.pop() == n
print(n)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
