## 문제

[BOJ 7510 - 고급 수학](https://www.acmicpc.net/problem/7510)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
for i in range(int(input())):
    li = sorted(map(int, input().split()))
    if li[0] ** 2 + li[1] ** 2 == li[2] ** 2:
        print(f"Scenario #{i+1}:")
        print("yes\n")
    else:
        print(f"Scenario #{i+1}:")
        print("no\n")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
