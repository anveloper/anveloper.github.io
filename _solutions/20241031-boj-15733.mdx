## 문제

[BOJ 15733 - 나는 누구인가](https://www.acmicpc.net/problem/15733)

정해진 문자열을 출력하라.

### 입력

입력 없음.

### 출력

"I'm Sexy"를 출력한다.

### 예제

| 입력   | 출력       |
| ------ | ---------- |
| (없음) | `I'm Sexy` |

## 풀이

고정 문자열을 출력한다.

1. "I'm Sexy"를 출력한다

**핵심 아이디어**: 단순 출력 문제이다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  cout << "I'm Sexy";
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
