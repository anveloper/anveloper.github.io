## 문제

[BOJ 15733 - 나는 누구인가](https://www.acmicpc.net/problem/15733)

> 정해진 문자열 "I'm Sexy"를 출력한다.

## 풀이

별도의 입력 없이 "I'm Sexy"를 출력하면 되는 단순 출력 문제이다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  cout << "I'm Sexy";
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
