## 문제

[BOJ 1362 - 펫](https://www.acmicpc.net/problem/1362)

> 반려동물의 초기 체중과 운동/식사 기록이 주어질 때, 최종 건강 상태를 판별한다.

## 풀이

운동(E)이면 체중을 줄이고 식사(F)이면 늘린다. 체중이 0 이하면 사망, 아니면 원래 체중의 0.5~2배 범위인지 확인한다.

## 코드

```javascript
const solution = (input) => {
  input = input.map((i) => i.split(" "));
  const sb = [];
  let SN = 0;
  while (++SN) {
    let [o, w] = input.shift().map(Number);
    if (o == 0 && w == 0) break;
    let isRip = false;
    while (true) {
      const [cmd, n] = input.shift();
      if (cmd === "#") break;
      else if (cmd === "E") { w -= +n; if (w <= 0) isRip = true; }
      else { w += +n; }
    }
    if (isRip) sb.push(`${SN} RIP`);
    else if (o / 2 < w && w < o * 2) sb.push(`${SN} :-)`);
    else sb.push(`${SN} :-(`);
  }
  return sb.join("\n");
};
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
