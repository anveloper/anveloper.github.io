## 문제

[BOJ 9070 - 장보기](https://www.acmicpc.net/problem/9070)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#define ll long long
using namespace std;

bool ChooseA(ll Ax, ll Ay, ll Bx, ll By)
{
  if (Ay * Bx != Ax * By)
    return Ay * Bx < Ax * By;
  return Ay < By;
}

int main(void)
{
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  ll T, N, i, a, b;

  cin >> T;
  while (T--)
  {
    cin >> N;

    ll x = 0, y = 2e9;
    for (i = 1; i <= N; i++)
    {
      cin >> a >> b;
      if (ChooseA(a, b, x, y))
        x = a, y = b;
    }

    cout << y << "\n";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
