## 문제

[BOJ 2183 - 테니스 시합](https://www.acmicpc.net/problem/2183)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int N;
string winners;
int main()
{
  cin >> N >> winners;
  cout << winners.back();
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
