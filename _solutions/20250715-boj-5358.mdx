## 문제

[BOJ 5358 - Football Team](https://www.acmicpc.net/problem/5358)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

for line in sys.stdin:
    line = line.rstrip()

    line = line.replace("i", "#")
    line = line.replace("e", "i")
    line = line.replace("#", "e")

    line = line.replace("I", "#")
    line = line.replace("E", "I")
    line = line.replace("#", "E")

    print(line)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
