## 문제

[BOJ 25206 - 너의 평점은](https://www.acmicpc.net/problem/25206)

> 20개 과목의 과목명, 학점, 등급이 주어졌을 때 전공 평점(가중 평균)을 계산한다.

## 풀이

각 과목의 등급을 숫자로 변환하여 학점과 곱한 값의 합을 총 학점으로 나눈다. P(Pass) 과목은 평점 계산에서 제외한다.

## 코드

```python
rating = ["A+", "A0", "B+", "B0", "C+", "C0", "D+", "D0", "F"]
grade = [4.5, 4.0, 3.5, 3.0, 2.5, 2.0, 1.5, 1.0, 0]

total = 0
result = 0
for _ in range(20):
    s, p, g = input().split()
    p = float(p)
    if g != "P":
        total += p
        result += p * grade[rating.index(g)]

print("%.6f" % (result / total))
```

## 복잡도

- 시간: O(1) (고정 20개 과목)
- 공간: O(1)
