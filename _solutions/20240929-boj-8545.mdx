## 문제

[BOJ 8545 - Zadanie próbne](https://www.acmicpc.net/problem/8545)

> 주어진 문자열을 뒤집어서 출력한다.

## 풀이

문자열을 입력받고 마지막 인덱스부터 0까지 역순으로 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
string s;
int main()
{
  cin >> s;
  for (int i = s.size() - 1; i >= 0; i--)
    cout << s[i];
}
```

## 복잡도

- 시간: O(N) (N: 문자열 길이)
- 공간: O(N)
