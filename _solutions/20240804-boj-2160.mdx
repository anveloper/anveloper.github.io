## 문제

[BOJ 2160 - 그림 비교](https://www.acmicpc.net/problem/2160)

> N개의 5x7 그림이 주어질 때, 서로 다른 칸의 수가 가장 적은 두 그림의 번호를 출력한다.

## 풀이

모든 그림 쌍에 대해 5x7 칸을 비교하여 다른 칸의 수를 세고, 최솟값을 가지는 쌍을 찾는다.

## 코드

```cpp
#include <stdio.h>

char arr[23][23][230];
int main()
{
  int n, ans, min = 36, mini = 0, minj = 0;
  scanf("%d", &n);
  for (int i = 1; i <= n; i++)
  {
    for (int k = 1; k <= 5; k++)
    {
      for (int j = 1; j <= 7; j++)
      {
        scanf(" %c", &arr[k][j][i]);
      }
    }
  }
  for (int i = 1; i <= n - 1; i++)
  {
    for (int j = i + 1; j <= n; j++)
    {
      ans = 0;
      for (int a = 1; a <= 5; a++)
      {
        for (int b = 1; b <= 7; b++)
        {
          if (arr[a][b][i] != arr[a][b][j])
          {
            ans++;
          }
        }
      }
      if (ans < min)
      {
        min = ans;
        mini = i;
        minj = j;
      }
    }
  }
  printf("%d %d", mini, minj);
}
```

## 복잡도

- 시간: O(N^2 * 35)
- 공간: O(N * 35)
