## 문제

[BOJ 1297 - TV 크기](https://www.acmicpc.net/problem/1297)

> TV 대각선 길이 D와 세로:가로 비율 H:W가 주어질 때, 실제 세로와 가로 길이를 구한다.

## 풀이

피타고라스 정리로 대각선과 비율 관계를 이용해 실제 가로·세로를 계산한다. q = sqrt(D^2 / (H^2 + W^2))로 비례 상수를 구한 뒤, height = floor(q * H), width = floor(q * W)를 출력한다.

## 코드

```javascript
const solution = (input) => {
  const [D, H, W] = input[0].trim().split(" ").map(Number);
  const q = Math.sqrt((D * D) / (H * H + W * W));
  const height = Math.floor(q * H);
  const width = Math.floor(q * W);
  return `${height} ${width}`;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
