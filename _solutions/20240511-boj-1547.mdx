## 문제

[BOJ 1547 - 공](https://www.acmicpc.net/problem/1547)

> 1번 컵에 공이 있을 때, M번의 컵 교환 후 공이 있는 컵 번호를 구한다.

## 풀이

공의 위치를 추적하며 교환 연산을 시뮬레이션한다.

## 코드

```javascript
const solution = (input) => {
  const M = input[0];
  let cup = 1;
  for (let i = 1; i <= M; i++) {
    let x = input[i].split(" ")[0], y = input[i].split(" ")[1];
    if (cup == x) cup = y; else if (cup == y) cup = x;
  }
  return cup;
};
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(M)
- 공간: O(1)
