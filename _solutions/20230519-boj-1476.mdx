## 문제

[BOJ 1476 - 날짜 계산](https://www.acmicpc.net/problem/1476)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```java
package day499;

import java.util.*;

public class Day467BOJ1476날짜계산 {

  public static void main(String args[]) {
    Scanner sc = new Scanner(System.in);
    int e = 1, s = 1, m = 1;
    int E = sc.nextInt();
    int S = sc.nextInt();
    int M = sc.nextInt();

    for (int year = 1;; year++) {
      if (e == E && s == S && m == M) {
        System.out.println(year);
        break;
      }
      e += 1;
      s += 1;
      m += 1;
      if (e == 16)
        e = 1;
      if (s == 29)
        s = 1;
      if (m == 20)
        m = 1;
    }
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
