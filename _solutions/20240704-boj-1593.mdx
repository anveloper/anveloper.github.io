## 문제

[BOJ 1593 - 문자 해독](https://www.acmicpc.net/problem/1593)

> 단어 W의 순열(애너그램)이 문자열 S에 부분 문자열로 몇 번 등장하는지 구한다.

## 풀이

길이가 g인 슬라이딩 윈도우를 문자열 S 위에서 이동시키며, 윈도우 내 문자 빈도 배열과 단어 W의 문자 빈도 배열이 일치하는지 비교한다. 알파벳 대소문자를 포함한 크기 58의 배열로 빈도를 관리한다.

## 코드

```python
g, s = map(int, input().split())
W = input()
S = input()

answer = 0
wa = [0] * 58
sa = [0] * 58

for x in W:
    wa[ord(x) - 65] += 1

start, length = 0, 0
for i in range(s):
    sa[ord(S[i]) - 65] += 1
    length += 1

    if length == g:
        if wa == sa:
            answer += 1
        sa[ord(S[start]) - 65] -= 1
        start += 1
        length -= 1

print(answer)
```

## 복잡도

- 시간: O(S * A) (A: 알파벳 크기 58, 배열 비교)
- 공간: O(A)
