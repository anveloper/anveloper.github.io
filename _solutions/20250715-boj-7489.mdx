## 문제

[BOJ 7489 - 팩토리얼](https://www.acmicpc.net/problem/7489)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys


def newfactorial(n):
    ans = 1
    for i in range(1, n + 1):
        ans *= i
        ans %= 1000000000000
        while ans % 10 == 0:
            ans /= 10
    return int(ans % 10)


for _ in range(int(sys.stdin.readline())):
    n = int(sys.stdin.readline())
    print(newfactorial(n))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
