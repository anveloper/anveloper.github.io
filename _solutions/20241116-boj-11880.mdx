## 문제

[BOJ 11880 - 개미](https://www.acmicpc.net/problem/11880)

> 세 정수가 주어질 때, 가장 큰 수의 제곱과 나머지 두 수의 합의 제곱을 더한 값을 구한다.

## 풀이

세 수 중 최댓값을 찾고, 나머지 두 수의 합을 구한다. 최댓값의 제곱 + 나머지 합의 제곱을 출력한다. 이는 직각삼각형의 빗변 제곱과 관련된 기하학적 계산이다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(NULL);

  int T;

  cin >> T;

  long long a, b, c, max_abc, sum;
  for (int i = 0; i < T; i++)
  {
    cin >> a >> b >> c;
    max_abc = max({a, b, c});

    if (max_abc == a)
    {
      sum = b + c;
    }
    else if (max_abc == b)
    {
      sum = a + c;
    }
    else
    {
      sum = a + b;
    }

    cout << max_abc * max_abc + sum * sum << "\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(T) (T: 테스트 케이스 수)
- 공간: O(1)
