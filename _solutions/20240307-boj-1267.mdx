## 문제

[BOJ 1267 - 핸드폰 요금](https://www.acmicpc.net/problem/1267)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```java
package day799;

import java.util.*;

public class Day765BOJ1267핸드폰요금 {

  public static void main(String args[]) throws Exception {
    Scanner sc = new Scanner(System.in);
    int N = sc.nextInt();
    int K;
    int YoungSik = 0, MinSik = 0;
    for (int i = 0; i < N; i++) {
      K = sc.nextInt();
      YoungSik += ((K / 30) + 1) * 10;
      MinSik += ((K / 60) + 1) * 15;

    }
    if (YoungSik == MinSik) {
      System.out.println("Y M " + YoungSik);
    } else if (YoungSik < MinSik) {
      System.out.println("Y " + YoungSik);
    } else if (YoungSik > MinSik) {
      System.out.println("M " + MinSik);
    }
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
