## 문제

[BOJ 2823 - 유턴 싫어](https://www.acmicpc.net/problem/2823)

## 풀이

문제의 조건에 맞게 풀이한다.

## 코드

```python
import sys
from collections import deque


def bfs(v):
    dx = [1, -1, 0, 0]
    dy = [0, 0, 1, -1]

    q = deque([v])

    while q:
        a, b = q.popleft()
        cnt = 0

        for i in range(4):
            x = a + dx[i]
            y = b + dy[i]
            if -1 < x < r and -1 < y < c and graph[x][y] == ".":
                cnt += 1
                if visited[x][y] == False:
                    visited[x][y] = True
                    q.append([x, y])
        if cnt < 2:
            return 1
    return 0


r, c = map(int, sys.stdin.readline().split())
chek = []

graph = []
for i in range(r):
    graph.append(list(map(str, sys.stdin.readline().strip())))
    for j in range(c):
        if graph[i][j] == ".":
            chek.append([i, j])

visited = [[False] * c for _ in range(r)]

print(bfs(chek[0]))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
