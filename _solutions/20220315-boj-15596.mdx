## 문제

[BOJ 15596 - 정수 N개의 합](https://www.acmicpc.net/problem/15596)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package com.ssafy.an.day049;

class Test { // 15596 정수N개의 합 함수 만들기 1단계....
	long sum(int[] a) {
		long sum = 0;
		for (int i = 0; i < a.length; i++) {
			sum += a[i];  
		}
		return sum;
	}
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
