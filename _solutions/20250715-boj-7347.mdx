## 문제

[BOJ 7347 - 플립과 시프트](https://www.acmicpc.net/problem/7347)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
t = int(input())
for _ in range(t):
    temp = list(map(int, input().split()))
    count, info = temp[0], temp[1:]
    if count % 2:
        print("YES")
    else:
        odd, even = 0, 0
        for idx, i in enumerate(info):
            if idx % 2:
                odd += i
            else:
                even += i
        if abs(odd - even) <= 1:
            print("YES")
        else:
            print("NO")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
