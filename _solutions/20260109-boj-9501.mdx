## 문제

[BOJ 9501 - 꿍의 우주여행](https://www.acmicpc.net/problem/9501)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int t;
  cin >> t;
  while (t--)
  {
    int n, d, ans = 0;
    cin >> n >> d;
    while (n--)
    {
      int vi, fi, ci;
      cin >> vi >> fi >> ci;
      if (vi * fi / ci >= d)
        ans++;
    }
    cout << ans << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
