## 문제

[BOJ 1598 - 꼬리를 무는 숫자 나열](https://www.acmicpc.net/problem/1598)

> 4행 무한열 격자에 1부터 순서대로 세로로 채울 때, 두 수의 위치 간 맨해튼 거리를 구한다.

## 풀이

각 수를 0-인덱스로 변환한 뒤, 행(나머지 4)과 열(몫 4)의 차이의 절대값을 더한다.

## 코드

```python
a,b=map(int,input().split())
a-=1;b-=1
print(abs(a//4-b//4)+abs(a%4-b%4))
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
