## 문제

[BOJ 1560 - 비숍](https://www.acmicpc.net/problem/1560)

## 풀이

문제의 조건에 맞게 풀이한다.

## 코드

```python
n = int(input())
if n == 1:
    print(1)
else:
    print(n * 2 - 2)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
