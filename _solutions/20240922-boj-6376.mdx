## 문제

[BOJ 6376 - e 계산](https://www.acmicpc.net/problem/6376)

> 0부터 9까지의 n에 대해 1/0! + 1/1! + ... + 1/n!의 누적합을 출력하여 자연상수 e를 근사한다.

## 풀이

팩토리얼 함수로 n!을 계산하고, 이전까지의 누적합에 1/n!을 더해가며 메모이제이션 배열에 저장한다. 0~2까지는 미리 계산된 값을 사용하고, 3~9는 반복문으로 계산한다.

## 코드

```cpp
#include <cstdio>

int factorial(int n)
{
  int ret = 1;
  for (int i = 1; i <= n; ++i)
  {
    ret *= i;
  }
  return ret;
}

double memo[10] = {1, 2, 2.5};
int main(void)
{
  printf("n e\n");
  printf("- -----------\n");
  printf("0 1\n");
  printf("1 2\n");
  printf("2 2.5\n");
  for (int i = 3; i < 10; ++i)
  {
    int f = factorial(i);
    memo[i] = memo[i - 1] + (double)1 / f;
    printf("%d %.9lf\n", i, memo[i]);
  }
  return 0;
}
```

## 복잡도

- 시간: O(N^2) (각 팩토리얼 계산에 O(N))
- 공간: O(N)
