## 문제

[BOJ 1263 - 시간 관리](https://www.acmicpc.net/problem/1263)

> N개의 작업이 각각 소요 시간과 마감 시간을 가질 때, 모든 작업을 마감 내에 끝내기 위한 가장 늦은 시작 시간을 구한다.

## 풀이

마감 시간 기준으로 내림차순 정렬 후, 뒤에서부터 작업을 배치한다. 현재 시작 가능한 시각은 마감 시간과 이전 시작 시각 중 작은 값에서 소요 시간을 뺀 것이다. 최종값이 음수면 불가능(-1)이다.

## 코드

```javascript
const solution = (input) => {
  const N = +input[0];
  const tasks = [];
  for (let i = 1; i <= N; i++) {
    const [t, s] = input[i].trim().split(" ").map(Number);
    tasks.push([t, s]);
  }

  tasks.sort((a, b) => b[1] - a[1]);

  let time = tasks[0][1] - tasks[0][0];
  for (let i = 1; i < N; i++) {
    time = Math.min(time, tasks[i][1]) - tasks[i][0];
  }

  return time < 0 ? -1 : time;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N log N)
- 공간: O(N)
