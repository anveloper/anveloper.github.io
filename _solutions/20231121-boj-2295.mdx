## 문제

[BOJ 2295 - 세 수의 합](https://www.acmicpc.net/problem/2295)

## 풀이

이분 탐색을 통해 최적의 값을 탐색한다.

## 코드

```java
package day699;

import java.io.*;
import java.util.*;

public class Day654BOJ2295세수의합 {
  public static void main(String[] args) throws Exception {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    int N, num[];

    N = Integer.parseInt(br.readLine());

    num = new int[N];

    for (int i = 0; i < N; i++) {
      num[i] = Integer.parseInt(br.readLine());
    }
    Arrays.sort(num);

    Set<Integer> set = new HashSet<>();
    for (int i = 0; i < N; i++) {
      for (int j = i; j < N; j++) {
        set.add(num[i] + num[j]);
      }
    }

    for (int i = N - 1; 0 <= i; i--) {
      for (int j = 0; j < N; j++) {

        if (set.contains(num[i] - num[j])) {
          System.out.println(num[i]);
          return;
        }
      }
    }
  }
}
```

## 복잡도

- 시간: O(N log N)
- 공간: O(N)
