## 문제

[BOJ 11943 - 파일 옮기기](https://www.acmicpc.net/problem/11943)

> 두 바구니에 사과와 귤이 섞여 있을 때, 한 바구니에 사과만, 다른 바구니에 귤만 모으려면 최소 몇 개를 옮겨야 하는지 구한다.

## 풀이

바구니1의 사과를 바구니2로 옮기고 바구니2의 귤을 바구니1로 옮기는 경우와, 그 반대 경우 중 이동 횟수가 적은 쪽을 선택한다. 첫 번째 바구니의 사과(a) + 두 번째 바구니의 귤(d) vs 첫 번째 바구니의 귤(b) + 두 번째 바구니의 사과(c) 중 최솟값을 출력한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
  ios::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);

  int a, b, c, d;
  cin >> a >> b >> c >> d;
  cout << min(a + d, b + c) << endl;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
