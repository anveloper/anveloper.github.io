## 문제

[BOJ 1440 - 타임머신](https://www.acmicpc.net/problem/1440)

> 콜론으로 구분된 세 수를 재배열하여 유효한 시각(HH:MM:SS)을 만들 수 있는 경우의 수를 구한다.

## 풀이

세 수의 모든 순열(최대 6가지)을 생성하여, 시(1~12), 분(0~59), 초(0~59) 범위를 만족하는 조합을 카운트한다.

## 코드

```python
t_machine = list(map(int, input().split(':')))
hour = [h for h in range(1, 13)]
min_sec = [ms for ms in range(60)]

result = 0

for i in range(3) :
    for j in range(3) :
        for k in range(3) :
            if i != j and j != k and k != i :
                if t_machine[i] in hour and t_machine[j] in min_sec and t_machine[k] in min_sec :
                    result += 1

print(result)
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
