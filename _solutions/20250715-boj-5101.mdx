## 문제

[BOJ 5101 - Sequences](https://www.acmicpc.net/problem/5101)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
while True:
    a, b, c = map(int, input().split())
    if a == b == c == 0:
        break
    if (c - a) % b == 0 and (c - a) // b >= 0:
        print((c - a) // b + 1)
    else:
        print("X")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
