## 문제

[BOJ 4446 - ROT13](https://www.acmicpc.net/problem/4446)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

vowels_lower = "aiyeou"
vowels_upper = "AIYEOU"
consonants_lower = "bkxznhdcwgpvjqtsrlmf"
consonants_upper = "BKXZNHDCWGPVJQTSRLMF"
mapping = {}

for seq, shift in [
    (vowels_lower, 3),
    (vowels_upper, 3),
    (consonants_lower, 10),
    (consonants_upper, 10),
]:
    n = len(seq)
    for i in range(n):
        mapping[seq[i]] = seq[(i + shift) % n]

trans = str.maketrans(mapping)

for line in sys.stdin:
    print(line.translate(trans), end="")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
