## 문제

[BOJ 1672 - DNA 해독](https://www.acmicpc.net/problem/1672)

> A, G, C, T로 이루어진 DNA 문자열이 주어지고, 두 문자의 결합 규칙표에 따라 뒤에서부터 하나의 문자로 축소한다.

## 풀이

4x4 결합 규칙 테이블을 딕셔너리로 정의한다. 문자열을 뒤집은 뒤 앞에서부터 순서대로 두 문자를 결합하며 하나의 문자가 남을 때까지 반복한다.

## 코드

```python
DNA = {
    "AA": "A",
    "AG": "C",
    "AC": "A",
    "AT": "G",
    "GA": "C",
    "GG": "G",
    "GC": "T",
    "GT": "A",
    "CA": "A",
    "CG": "T",
    "CC": "C",
    "CT": "G",
    "TA": "G",
    "TG": "A",
    "TC": "G",
    "TT": "T",
}
N = int(input())
S = input().rstrip()
S = "".join(reversed(list(S)))
tmp = S[0]
for i in range(1, N):
    tmp = DNA[tmp + S[i]]
print(tmp)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
