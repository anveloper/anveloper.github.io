## 문제

[BOJ 2386 - 도비의 영어 공부](https://www.acmicpc.net/problem/2386)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
while True:
    n = input()
    if n == "#":
        break
    value = n[0]
    data = n[2::]
    result = data.count(value) + data.count(value.upper())
    print(value, result)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
