## 문제

[BOJ 5724 - 파인만](https://www.acmicpc.net/problem/5724)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int n;
  while (1)
  {
    int ans = 0;
    cin >> n;
    if (!n)
      break;
    ans += 1 + n * n;
    for (int i = 2; i <= n - 1; i++)
      ans += i * i;
    if (n != 1)
      cout << ans << '\n';
    else
      cout << 1 << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
