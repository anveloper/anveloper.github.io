## 문제

[BOJ 10926 - ??!](https://www.acmicpc.net/problem/10926)

> 아이디를 입력받아 뒤에 "??!"를 붙여 출력한다.

## 풀이

문자열을 입력받아 "??!"를 이어붙인 뒤 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(int argc, char const *argv[])
{

  string id;
  cin >> id;

  id += "??!";
  cout << id;

  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(N) (N: 아이디 길이)
