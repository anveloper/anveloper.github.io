## 문제

[BOJ 11383 - 뚊](https://www.acmicpc.net/problem/11383)

> N x M 크기의 그림의 각 문자를 두 번 반복하여 N x 2M 크기로 만든 결과가 주어진 그림과 같은지 확인한다.

## 풀이

N줄의 원본 문자열을 읽으면서 각 문자를 두 번 반복한 문자열을 만든다. 이후 N줄의 비교 대상 문자열을 읽어 순서대로 비교하여, 하나라도 다르면 "Not Eyfa", 모두 같으면 "Eyfa"를 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <vector>

using namespace std;

int main()
{
  int n, m;
  cin >> n >> m;
  string word;
  vector<string> words;
  for (int i = 0; i < n; i++)
  {
    cin >> word;

    string c = "";
    for (int j = 0; j < m; j++)
    {
      c += word[j];
      c += word[j];
    }

    words.push_back(c);
  }

  bool flag = 0;
  string compare;
  for (auto &ele : words)
  {
    cin >> compare;
    if (ele != compare)
    {
      flag = 1;
      break;
    }
  }

  if (flag)
  {
    cout << "Not Eyfa";
  }
  else
  {
    cout << "Eyfa";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N * M)
- 공간: O(N * M)
