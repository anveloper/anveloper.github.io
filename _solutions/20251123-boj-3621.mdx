## 문제

[BOJ 3621 - 족보](https://www.acmicpc.net/problem/3621)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);

  int n,
      d;
  cin >> n >> d;
  vector<int> cnt(n + 1);
  for (int i = 0; i < n; i++)
  {
    int p;
    cin >> p;
    cnt[p]++;
  }

  int ans = 0;
  for (int i = 0; i <= n; i++)
    if (cnt[i] > d)
      ans += (cnt[i] - 2) / (d - 1);
  cout << ans;

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
