## 문제

[BOJ 6190 - Another Cow Number Game](https://www.acmicpc.net/problem/6190)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#define ll long long
using namespace std;
ll n, score;

int main()
{
  cin >> n;
  while (n != 1)
  {
    if (n % 2)
      n = 3 * n + 1;
    else
      n /= 2;
    score++;
  }
  cout << score;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
