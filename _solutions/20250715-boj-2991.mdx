## 문제

[BOJ 2991 - 사나운 개](https://www.acmicpc.net/problem/2991)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
A, B, C, D = map(int, input().split())
delivery = list(map(int, input().split()))

for i in delivery:
    dog = 0
    if 0 < i % (A + B) <= A:
        dog += 1
    if 0 < i % (C + D) <= C:
        dog += 1

    print(dog)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
