## 문제

[BOJ 6378 - 디지털 루트](https://www.acmicpc.net/problem/6378)

> 주어진 수의 각 자릿수를 더하는 과정을 한 자리 수가 될 때까지 반복하여 디지털 루트를 구한다.

## 풀이

큰 수가 입력될 수 있으므로 문자열로 받아 각 자릿수를 더한다. 합이 10 이상이면 다시 문자열로 변환하여 반복한다. 한 자리 수가 되면 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int main(int argc, const char *argv[])
{
  string s;
  int sum = 0;
  while (1)
  {
    cin >> s;
    if (s == "0")
      break;

    sum = 0;
    while (1)
    {
      for (int i = 0; i < s.length(); i++)
        sum += s[i] - '0';

      if (sum < 10)
        break;
      else
      {
        s = to_string(sum);
        sum = 0;
      }
    }
    cout << sum << '\n';
  }
  return 0;
}
```

## 복잡도

- 시간: O(T * log N) (T: 테스트 케이스 수, N: 입력 수의 크기)
- 공간: O(log N)
