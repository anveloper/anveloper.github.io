## 문제

[BOJ 3060 - 욕심쟁이 돼지](https://www.acmicpc.net/problem/3060)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
t = int(input())
for _ in range(t):
    n = int(input())
    line = list(map(int, input().split()))
    total = sum(line)
    count = 0
    while n >= total:
        count += 1
        total *= 4
    print(count + 1)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
