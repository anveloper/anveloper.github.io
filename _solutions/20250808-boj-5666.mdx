## 문제

[BOJ 5666 - Hot Dogs](https://www.acmicpc.net/problem/5666)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int h, p;

  while (scanf("%d %d", &h, &p) != EOF)
  {
    double ans = (double)h / (double)p;
    printf("%.2f\n", ans);
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
