## 문제

[BOJ 9226 - 도깨비말](https://www.acmicpc.net/problem/9226)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{

  while (1)
  {
    string word;

    cin >> word;
    if (word == "#")
      break;

    int start = 0;
    bool flag = 0;
    int word_len = (int)word.length();
    while (1)
    {
      if (start > word_len + 1)
      {
        flag = 1;
        break;
      }
      if (word[start] == 'a' || word[start] == 'e' || word[start] == 'i' || word[start] == 'o' || word[start] == 'u')
        break;
      else
      {
        word += word[start];
        start++;
      }
    }

    if (flag)
    {
      for (int i = 0; i < word_len; i++)
      {
        cout << word[i];
      }
    }
    else
    {
      int len = (int)word.length();
      for (int i = start; i < len; i++)
      {
        cout << word[i];
      }
    }

    cout << "ay\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
