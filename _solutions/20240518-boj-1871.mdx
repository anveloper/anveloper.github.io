## 문제

[BOJ 1871 - 좋은 자동차 번호판](https://www.acmicpc.net/problem/1871)

> LLL-DDDD 형태의 번호판에서 문자부(26진수)와 숫자부의 차이가 100 이하이면 nice, 아니면 not nice를 출력한다.

## 풀이

문자 3자리를 26진수로 변환하고, 숫자 4자리와의 차이의 절대값을 비교한다.

## 코드

```python
for _ in range(int(input())):
  L, D = input().split('-')
  n = int(D)
  s = 0
  for i in range(3):
    s += (ord(L[i])-65) * 26**(2-i)
  print("nice" if abs(s-n) <= 100 else "not nice")
```

## 복잡도

- 시간: O(T) — T는 테스트 케이스 수
- 공간: O(1)
