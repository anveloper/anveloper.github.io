## 문제

[BOJ 1622 - 공통 순열](https://www.acmicpc.net/problem/1622)

> 두 문자열이 주어졌을 때, 두 문자열 모두의 순열의 부분 문자열이 될 수 있는 가장 긴 문자열을 구한다.

## 풀이

두 문자열에서 공통으로 등장하는 문자를 찾고, 각 문자의 최소 등장 횟수만큼 해당 문자를 결과에 포함시킨다. 결과를 알파벳 순으로 정렬하여 출력한다.

## 코드

```python
while True:
    try:
        a = input()
        b = input()

        a_set = set(a)
        b_set = set(b)

        x = a_set & b_set

        answer = []
        for w in x:
            a_count = a.count(w)
            b_count = b.count(w)
            answer.append(w * min(a_count, b_count))

        answer.sort()
        print("".join(answer))
    except:
        break
```

## 복잡도

- 시간: O(T * (A + B)) (T는 테스트 케이스 수, A, B는 문자열 길이)
- 공간: O(A + B)
