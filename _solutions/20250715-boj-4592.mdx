## 문제

[BOJ 4592 - 중복을 없애자](https://www.acmicpc.net/problem/4592)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
while True:
    numbers = input()
    result = ""
    if numbers == "0":
        break
    else:
        numbers = numbers.split(" ")
        N = int(numbers[0])
        numbers = numbers[1:]
        result += numbers[0] + " "
        for index in range(1, N):
            if numbers[index - 1] != numbers[index]:
                result += numbers[index] + " "
        print(f"{result}$")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
