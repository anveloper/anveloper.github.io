## 문제

[BOJ 2985 - 세 수](https://www.acmicpc.net/problem/2985)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
a, b, c = map(int, input().split())
if a + b == c:
    print(f"{a}+{b}={c}")
elif a - b == c:
    print(f"{a}-{b}={c}")
elif a * b == c:
    print(f"{a}*{b}={c}")
elif a // b == c:
    print(f"{a}/{b}={c}")
elif a == b + c:
    print(f"{a}={b}+{c}")
elif a == b - c:
    print(f"{a}={b}-{c}")
elif a == b * c:
    print(f"{a}={b}*{c}")
elif a == b // c:
    print(f"{a}={b}/{c}")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
