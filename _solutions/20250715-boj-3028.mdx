## 문제

[BOJ 3028 - 창영마을](https://www.acmicpc.net/problem/3028)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
li = [1, 0, 0]
s = input()
for c in s:
    if c == "A":
        li[0], li[1] = li[1], li[0]
    elif c == "B":
        li[1], li[2] = li[2], li[1]
    else:
        li[0], li[2] = li[2], li[0]
print(li.index(1) + 1)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
