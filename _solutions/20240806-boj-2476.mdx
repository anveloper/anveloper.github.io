## 문제

[BOJ 2476 - 주사위 게임](https://www.acmicpc.net/problem/2476)

> 주사위 3개를 던져 같은 눈의 수에 따라 상금이 결정되는 규칙에서, 여러 참가자 중 최대 상금을 구한다.

## 풀이

3개 모두 같으면 10000 + 같은눈*1000, 2개만 같으면 1000 + 같은눈*100, 모두 다르면 가장 큰 눈*100으로 상금을 계산한 뒤 최댓값을 출력한다.

## 코드

```cpp
#include <stdio.h>

int main()
{
  int x, y, z, prize[1000], max = 0, n, i, sum = 0, big = 0;
  scanf("%d", &n);

  for (i = 0; i < n; i++)
  {
    scanf("%d %d %d", &x, &y, &z);
    if (x == y && x == z && y == z)
      prize[i] = 10000 + x * 1000;
    else if (x == y || x == z)
      prize[i] = 1000 + x * 100;
    else if (y == z)
      prize[i] = 1000 + z * 100;
    else
    {
      if (x > y && x > z)
        max = x;
      else if (y > x && y > z)
        max = y;
      else if (z > x && z > y)
        max = z;
      prize[i] = max * 100;
    }
    if (prize[i] > big)
      big = prize[i];
  }
  printf("%d", big);
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
