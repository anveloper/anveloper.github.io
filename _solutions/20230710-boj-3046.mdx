## 문제

[BOJ 3046 - R2](https://www.acmicpc.net/problem/3046)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day599;

import java.util.Scanner;

public class Day583BOJ3046R2 {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int a = sc.nextInt();
    int b = sc.nextInt();
    System.out.println((b * 2) - a);
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
