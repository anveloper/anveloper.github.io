## 문제

[BOJ 2607 - 비슷한 단어](https://www.acmicpc.net/problem/2607)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
N = int(input())
target = list(input()) # 비교 대상 단어(첫 단어)
answer = 0

for _ in range(N-1):
    compare = target[:] 
    word = input() # 새로운 단어
    cnt = 0

    for w in word:
        if w in compare:
            compare.remove(w)
        else:
            cnt += 1

    if cnt < 2 and len(compare) < 2:
        answer += 1

print(answer)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
