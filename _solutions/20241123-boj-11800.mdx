## 문제

[BOJ 11800 - Tawla](https://www.acmicpc.net/problem/11800)

> 두 주사위의 눈이 주어질 때, 아랍어 규칙에 따라 해당하는 이름을 출력한다.

## 풀이

주사위 눈 1~6에 대응하는 아랍어 이름 배열과, 두 눈이 같을 때의 이름 배열을 미리 정의한다. 두 눈이 같으면 동일 눈 이름을, 5와 6이면 특수 이름을, 그 외에는 큰 눈의 이름 + 작은 눈의 이름 순서로 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int n, a, b;
string diceNumName[7] = {"", "Yakk", "Doh", "Seh", "Ghar", "Bang", "Sheesh"};
string sameDiceNumName[7] = {"", "Habb Yakk", "Dobara", "Dousa", "Dorgy", "Dabash", "Dosh"};
int main()
{
  cin >> n;
  for (int i = 1; i <= n; i++)
  {
    cin >> a >> b;
    cout << "Case " << i << ": ";
    int small = min(a, b);
    int big = max(a, b);
    if (small == big)
    {
      cout << sameDiceNumName[small] << '\n';
      continue;
    }
    if (small == 5 && big == 6)
    {
      cout << "Sheesh Beesh\n";
      continue;
    }
    cout << diceNumName[big] << ' ' << diceNumName[small] << '\n';
  }
}
```

## 복잡도

- 시간: O(N) (N: 테스트 케이스 수)
- 공간: O(1)
