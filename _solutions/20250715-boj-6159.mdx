## 문제

[BOJ 6159 - Costume Party](https://www.acmicpc.net/problem/6159)

## 풀이

정렬을 활용하여 문제를 해결한다. 투 포인터 기법을 활용한다.

## 코드

```python
import sys

input = sys.stdin.readline
n, s = map(int, input().split())
arr = sorted([int(input()) for _ in range(n)])

ans = 0
j = n - 1

for i in range(n):
    while arr[i] + arr[j] > s and i < j:
        j -= 1
    if i == j:
        break
    ans += j - i
print(ans)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
