## 문제

[BOJ 6438 - Reverse Text](https://www.acmicpc.net/problem/6438)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

int main()
{
  int N;
  cin >> N;
  cin.ignore();
  for (int i = 0; i < N; i++)
  {
    string temp;
    getline(cin, temp);
    reverse(temp.begin(), temp.end());
    cout << temp << "\n";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
