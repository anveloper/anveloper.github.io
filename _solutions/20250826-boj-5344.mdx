## 문제

[BOJ 5344 - GCD](https://www.acmicpc.net/problem/5344)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;
typedef long long ll;

ll gcd(ll x, ll y)
{
  if (y == 0)
    return x;
  return gcd(y, x % y);
}

int T;

int main()
{
  ios::sync_with_stdio(0);
  cin.tie(0);

  cin >> T;
  while (T--)
  {
    ll x, y;
    cin >> x >> y;
    cout << gcd(x, y) << '\n';
  }
}
```

## 복잡도

- 시간: O(log N)
- 공간: O(1)
