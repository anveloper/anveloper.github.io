## 문제

[BOJ 2428 - 표절](https://www.acmicpc.net/problem/2428)

## 풀이

이분 탐색을 통해 최적의 값을 탐색한다.

## 코드

```python
N = int(input())
li = sorted(map(int, input().split()))
res = 0
for i in range(N - 1):
    s, e = i + 1, N - 1
    t = -1
    while s <= e:
        m = (s + e) // 2
        if li[i] >= 0.9 * li[m]:
            t = m
            s = m + 1
        else:
            e = m - 1
    res += t - i if t > -1 else 0
print(res)
```

## 복잡도

- 시간: O(N log N)
- 공간: O(N)
