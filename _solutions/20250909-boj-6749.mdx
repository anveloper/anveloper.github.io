## 문제

[BOJ 6749 - Next in line](https://www.acmicpc.net/problem/6749)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(void)
{
  int a, b, differ;
  cin >> a >> b;
  differ = b - a;
  cout << b + differ;
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
