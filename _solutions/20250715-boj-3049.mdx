## 문제

[BOJ 3049 - 다각형의 대각선](https://www.acmicpc.net/problem/3049)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n = int(input())
print(int(n * (n - 1) * (n - 2) * (n - 3) / 24))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
