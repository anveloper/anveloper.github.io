## 문제

[BOJ 1975 - Number Game](https://www.acmicpc.net/problem/1975)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys

input = sys.stdin.readline


def func(x, b):
    if x != 0 and x % b == 0:
        return 1 + func(x // b, b)
    else:
        return 0


c = [0] * 1001
for x in range(2, 1001):
    c[x] = sum(func(x, b) for b in range(2, x + 1))

for _ in range(int(input())):
    x = int(input())
    print(c[x])
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
