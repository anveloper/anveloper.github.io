## 문제

[BOJ 6856 - Roll the Dice](https://www.acmicpc.net/problem/6856)

## 풀이

가능한 모든 경우를 탐색하여 답을 구한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m, n;
  cin >> m >> n;

  int ways = 0;
  for (int i = 1; i <= m; i++)
  {
    if (10 - i >= 1 && 10 - i <= n)
      ways++;
  }

  if (ways == 1)
    cout << "There is 1 way to get the sum 10." << "\n";
  else
    cout << "There are " << ways << " ways to get the sum 10." << "\n";

  return 0;
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
