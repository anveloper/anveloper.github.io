## 문제

[BOJ 4285 - Prerequisites?](https://www.acmicpc.net/problem/4285)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main() {
  int n, m, num, a, b, c;
  while (cin >> n >> m) {
    bool arr[10000] = { 0 };
    while (n --) {
      cin >> num;
      arr[num] = 1;
    }
    string answer("yes\n");
    while (m --) {
      cin >> a >> b;
      while (a --) {
        cin >> c;
        if (arr[c] == 1) b --;
      }
      if (0 < b) answer = "no\n";
    }
    cout << answer;
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
