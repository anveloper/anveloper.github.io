## 문제

[BOJ 5523 - 경기 결과](https://www.acmicpc.net/problem/5523)

> N번의 경기에서 각 경기의 A, B 점수를 비교하여 A의 총 승수와 B의 총 승수를 출력한다.

## 풀이

N번 반복하면서 각 경기의 A, B 점수를 비교한다. A가 크면 A_count를, B가 크면 B_count를 증가시키고, 최종 결과를 출력한다.

## 코드

```cpp
#include <stdio.h>
int main()
{
  int N, A, B, A_count = 0, B_count = 0;
  scanf("%d", &N);
  for (int i = 0; i < N; i++)
  {
    scanf("%d %d", &A, &B);
    if (A > B)
      A_count++;
    else if (A < B)
      B_count++;
  }
  printf("%d %d", A_count, B_count);
}
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
