## 문제

[BOJ 5246 - Checkerboard Rows](https://www.acmicpc.net/problem/5246)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <vector>
#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int cntBoard;
  cin >> cntBoard;

  for (int i = 0; i < cntBoard; i++)
  {
    int cntPieces;
    cin >> cntPieces;
    vector<int> rows(8, 0);
    for (int j = 0; j < cntPieces; j++)
    {
      int x, y;
      cin >> x >> y;

      rows[y - 1]++;
    }

    int maxPieces = *max_element(rows.begin(), rows.end());
    cout << maxPieces << "\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
