## 문제

[BOJ 3486 - Adding Reversed Numbers](https://www.acmicpc.net/problem/3486)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
  int t;
  cin >> t;
  while (t--)
  {
    string a, b, c;
    cin >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    c = to_string(stoi(a) + stoi(b));
    reverse(c.begin(), c.end());
    cout << stoi(c) << "\n";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
