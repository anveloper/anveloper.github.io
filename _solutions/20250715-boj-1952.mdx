## 문제

[BOJ 1952 - 달팽이2](https://www.acmicpc.net/problem/1952)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
m, n = map(int, input().split())
print(2 * min(m, n) - 1 - (m <= n))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
