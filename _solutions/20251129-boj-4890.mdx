## 문제

[BOJ 4890 - Tiles of Tetris, NOT!](https://www.acmicpc.net/problem/4890)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;
int main(void)
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);
  while (1)
  {
    long long W, H;
    cin >> W >> H;
    if (W == 0 && H == 0)
    {
      break;
    }
    if (W < H)
    {
      swap(W, H);
    }
    if (W == H)
    {
      cout << 1 << "\n";
      continue;
    }
    if (W % H == 0)
    {
      cout << W / H << "\n";
    }
    else
    {
      cout << W * H << "\n";
    }
  }
  return 0;
}
```

## 복잡도

- 시간: O(log N)
- 공간: O(1)
