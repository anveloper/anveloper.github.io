## 문제

[BOJ 5586 - JOI와 IOI](https://www.acmicpc.net/problem/5586)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

s = sys.stdin.readline().strip()

joi_count = 0
ioi_count = 0

for i in range(len(s) - 2):
    segment = s[i : i + 3]
    if segment == "JOI":
        joi_count += 1
    elif segment == "IOI":
        ioi_count += 1

print(joi_count)
print(ioi_count)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
