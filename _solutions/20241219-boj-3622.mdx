## 문제

[BOJ 3622 - 어떤 호박의 할로윈 여행](https://www.acmicpc.net/problem/3622)

> 호박의 무게와 가격 등 여러 조건을 분기하여 구매 가능 여부를 판별한다.

## 풀이

A의 가격과 허용 무게, B의 가격과 허용 무게, 그리고 소지금 P가 주어진다. 두 물건의 가격 합이 P 이하이거나, 한쪽이 다른 쪽의 허용 무게 이내이면서 나머지 가격이 P 이하인 경우 등 여러 조건을 분기하여 Yes/No를 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);
  int A, a, B, b, P;
  cin >> A >> a >> B >> b >> P;
  cout << ((A + B <= P) || (B <= P && A <= b) || (A <= P && B <= a) ? "Yes" : "No");
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
