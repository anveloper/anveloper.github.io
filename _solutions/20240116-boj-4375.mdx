## 문제

[BOJ 4375 - 1](https://www.acmicpc.net/problem/4375)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```java
package day749;

import java.util.*;

public class Day714BOJ4375일 {

  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);

    while (sc.hasNextInt()) {
      int num = sc.nextInt();
      int mod = 0;

      for (int i = 1;; i++) {
        mod = (mod * 10 + 1) % num;
        if (mod == 0) {
          System.out.println(i);
          break;
        }
      }
    }
    sc.close();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
