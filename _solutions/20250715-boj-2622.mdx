## 문제

[BOJ 2622 - 삼각형만들기](https://www.acmicpc.net/problem/2622)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n, r = int(input()), 0
for i in range((n - 1) // 3 + 1, (n - 1) // 2 + 1):
    a = n - i
    cnt = min(i, a - 1) - max(1, a - i) + 1
    if cnt <= 0:
        continue
    r += cnt // 2 + cnt % 2
print(r)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
