## 문제

[BOJ 3733 - Shares](https://www.acmicpc.net/problem/3733)

N명의 직원과 사장 1명이 S개의 주식을 균등 분배할 때, 한 사람이 받는 주식 수를 구하라.

### 입력

여러 줄에 N과 S가 주어진다 (EOF까지).

### 출력

각 줄에 한 사람의 몫을 출력한다.

### 예제

| 입력             | 출력       |
| ---------------- | ---------- |
| `1 100` `3 1000` | `50` `250` |

## 풀이

총 인원 N+1로 나누어 한 사람의 몫을 구한다.

1. `while(cin >> n >> s)` 패턴으로 EOF까지 입력을 처리한다
2. 각 테스트 케이스에서 `S / (N + 1)`을 출력한다

**핵심 아이디어**: 단순 나눗셈 문제이며, EOF 처리가 핵심이다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, s;
int main()
{
  while (cin >> n >> s)
    cout << s / (n + 1) << '\n';
}
```

## 복잡도

- 시간: O(T) (T: 테스트 케이스 수)
- 공간: O(1)
