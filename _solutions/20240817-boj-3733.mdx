## 문제

[BOJ 3733 - Shares](https://www.acmicpc.net/problem/3733)

> N명의 직원과 사장 1명이 S개의 주식을 균등 분배할 때, 한 사람이 받는 주식 수를 구한다.

## 풀이

총 인원은 N+1명이므로 `S / (N + 1)`을 계산하면 된다. 여러 테스트 케이스가 EOF까지 주어지므로 `while(cin >> n >> s)` 패턴으로 입력을 처리한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, s;
int main()
{
  while (cin >> n >> s)
    cout << s / (n + 1) << '\n';
}
```

## 복잡도

- 시간: O(T) (T: 테스트 케이스 수)
- 공간: O(1)
