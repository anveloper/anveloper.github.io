## 문제

[BOJ 6721 - Backward numbers](https://www.acmicpc.net/problem/6721)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int reverse(int x)
{
  int ret = 0;
  while (x)
  {
    ret = ret * 10 + x % 10;
    x /= 10;
  }
  return ret;
}

void solve(void)
{
  int a, b;
  cin >> a >> b;

  cout << reverse(reverse(a) + reverse(b)) << "\n";
}

int main(void)
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  cin >> n;
  while (n--)
    solve();
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
