## 문제

[BOJ 5338 - 마이크로소프트 로고](https://www.acmicpc.net/problem/5338)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day699;

public class Day675BOJ5338마이크로소프트로고 {
  public static void main(String[] args) {
    StringBuilder sb = new StringBuilder();
    sb.append("       _.-;;-._\n"
        + "'-..-'|   ||   |\n"
        + "'-..-'|_.-;;-._|\n"
        + "'-..-'|   ||   |\n"
        + "'-..-'|_.-''-._|");
    System.out.println(sb);
  }
}
```

- 시간: O(N)
- 공간: O(N)
