## 문제

[BOJ 8719 - Piłeczka](https://www.acmicpc.net/problem/8719)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int t, x, w, ans;

int main()
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);

  cin >> t;
  while (t--)
  {
    ans = 0;
    cin >> x >> w;
    while (x < w)
    {
      x *= 2;
      ans++;
    }
    cout << ans << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
