## 문제

[BOJ 1864 - 문어 숫자](https://www.acmicpc.net/problem/1864)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
if __name__ == "__main__":
    wave = {"-": 0, "\\": 1, "(": 2, "@": 3, "?": 4, ">": 5, "&": 6, "%": 7, "/": -1}
    while True:
        n = input()
        if n == "#":
            break

        result = 0
        for i in range(len(n)):
            result += wave[n[i]] * 8 ** (len(n) - i - 1)
        print(result)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
