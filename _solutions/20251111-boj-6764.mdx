## 문제

[BOJ 6764 - Sounds fishy!](https://www.acmicpc.net/problem/6764)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int a[4];
int main()
{
  for (int i = 0; i < 4; i++)
    cin >> a[i];
  if (a[0] > a[1] && a[1] > a[2] && a[2] > a[3])
    cout << "Fish Diving";
  else if (a[0] < a[1] && a[1] < a[2] && a[2] < a[3])
    cout << "Fish Rising";
  else if (a[0] == a[1] && a[1] == a[2] && a[2] == a[3])
    cout << "Fish At Constant Depth";
  else
    cout << "No Fish";
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
