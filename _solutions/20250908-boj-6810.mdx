## 문제

[BOJ 6810 - ISBN](https://www.acmicpc.net/problem/6810)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int a, b, c;

int main()
{
  cin >> a >> b >> c;
  cout << "The 1-3-sum is " << 91 + a * 1 + b * 3 + c * 1;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
