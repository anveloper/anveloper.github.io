## 문제

[BOJ 12790 - Mini Fantasy War](https://www.acmicpc.net/problem/12790)

> 캐릭터의 기본 스탯 4개와 장비 보너스 4개가 주어질 때 공식에 따라 전투력을 계산한다.

## 풀이

기본 스탯과 장비 보너스를 합산하되, HP와 MP는 최소 1, 공격력은 최소 0으로 보정한다. 전투력 = HP × 1 + MP × 5 + 공격력 × 2 + 방어력 × 2 공식으로 계산하여 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int tc;
  cin >> tc;

  int a[8];
  for (int j = 0; j < tc; j++)
  {
    for (int i = 0; i < 8; i++)
      cin >> a[i];
    a[0] += a[4];
    a[1] += a[5];
    a[2] += a[6];
    a[3] += a[7];

    a[0] < 1 ? a[0] = 1 : 0;
    a[1] < 1 ? a[1] = 1 : 0;
    a[2] < 0 ? a[2] = 0 : 0;

    cout << a[0] * 1 + a[1] * 5 + a[2] * 2 + a[3] * 2 << "\n";
  }
}
```

## 복잡도

- 시간: O(T)
- 공간: O(1)
