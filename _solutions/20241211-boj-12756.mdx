## 문제

[BOJ 12756 - 고급 여관](https://www.acmicpc.net/problem/12756)

> 두 플레이어가 동시에 공격하여 체력이 0 이하가 되면 전투가 끝난다. 승자를 판별한다.

## 풀이

매 턴 A의 체력에서 B의 공격력을 빼고, B의 체력에서 A의 공격력을 빼는 시뮬레이션을 반복한다. 둘 다 동시에 0 이하가 되면 DRAW, 하나만 0 이하가 되면 상대 플레이어가 승리한다.

## 코드

```cpp
#include <stdio.h>
#include <stdlib.h>
int main()
{
  int a, b, c, d;
  scanf("%d %d %d %d", &a, &b, &c, &d);

  while (1)
  {
    b = b - c;
    d = d - a;
    if (b <= 0 || d <= 0)
      break;
  }

  if (b <= 0 && d <= 0)
    printf("DRAW\n");
  else if (d <= 0)
    printf("PLAYER A\n");
  else if (b <= 0)
    printf("PLAYER B\n");
}
```

## 복잡도

- 시간: O(min(B/C, D/A))
- 공간: O(1)
