## 문제

[BOJ 1568 - 새](https://www.acmicpc.net/problem/1568)

> N마리의 새가 전깃줄에 앉아 있을 때, 노래에 따라 1, 2, 3, ... 마리씩 날아간다. 남은 새보다 많으면 1부터 다시 센다.

## 풀이

카운터를 1부터 증가시키며 빼고, 남은 수보다 크면 1로 리셋한다.

## 코드

```javascript
const solution = (input) => {
  let N = Number(input[0]), sec = 1, cnt = 0;
  while (N) { N -= sec++; cnt++; if (sec > N) sec = 1; }
  return cnt;
};
const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(sqrt(N))
- 공간: O(1)
