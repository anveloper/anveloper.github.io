## 문제

[BOJ 1347 - 미로 만들기](https://www.acmicpc.net/problem/1347)

> R(오른쪽 회전), L(왼쪽 회전), F(전진) 명령어를 순서대로 실행하여 방문한 칸을 '.'으로, 나머지를 '#'으로 출력한다.

## 풀이

초기 방향을 남쪽으로 설정하고 명령어에 따라 방향을 변경하거나 전진한다. 방문한 모든 좌표를 기록한 뒤, 좌표의 최솟값을 기준으로 보정하여 2차원 배열에 '.'과 '#'을 채워 출력한다.

## 코드

```python
import sys

length = int(sys.stdin.readline())
note = list(sys.stdin.readline().rstrip())

dx = [0, 1, 0, -1]
dy = [1, 0, -1, 0]

x, y, direction = 0, 0, 1
graph = [[x, y]]

for i in note:
    if i == "R":
        direction = (direction + 1) % 4
    if i == "L":
        direction = (direction - 1) % 4
    if i == "F":
        x = x + dx[direction]
        y = y + dy[direction]
        graph.append([x, y])

n = max(graph)[0] - min(graph)[0] + 1

m = max(graph, key=lambda x: x[1])[1] - min(graph, key=lambda x: x[1])[1] + 1

min_x = min(graph, key=lambda x: x[0])[0]
min_y = min(graph, key=lambda x: x[1])[1]

for i in graph:
    i[0] = i[0] + abs(min_x)
    i[1] = i[1] + abs(min_y)


map = [[False] * m for _ in range(n)]

for i in graph:
    if map[i[0]][i[1]] == False:
        map[i[0]][i[1]] = "."

for i in map:
    for j in range(m):
        if i[j] == False:
            i[j] = "#"

for i in map:
    for j in range(len(i)):
        print(i[j], end="")
    print("")
```

## 복잡도

- 시간: O(N) (N은 명령어 길이)
- 공간: O(N)
