## 문제

[BOJ 15962 - 새로운 시작](https://www.acmicpc.net/problem/15962)

> "파이팅!!"을 출력한다.

## 풀이

고정 문자열 "파이팅!!"을 그대로 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  cout << "파이팅!!";
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
