## 문제

[BOJ 1673 - 치킨 쿠폰](https://www.acmicpc.net/problem/1673)

> 치킨 n마리를 사면 쿠폰 n장을 받고, 쿠폰 k장을 모으면 치킨 1마리와 교환할 수 있을 때 총 먹을 수 있는 치킨 수를 구한다.

## 풀이

처음 n마리에서 받는 쿠폰으로 시작하여, 쿠폰 k장당 치킨 1마리를 교환하는 과정을 반복한다. 교환한 치킨에서도 쿠폰이 나오므로 남은 쿠폰(n//k)과 나머지(n%k)를 합산하며 진행한다.

## 코드

```python
while True:
    try:
        y = 0
        n, k = map(int, input().split())
        y += n
        while n // k:
            y += n // k
            n = n // k + n % k
        print(y)
    except:
        break
```

## 복잡도

- 시간: O(log_k(N))
- 공간: O(1)
