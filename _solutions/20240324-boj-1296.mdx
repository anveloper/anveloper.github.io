## 문제

[BOJ 1296 - 팀 이름 정하기](https://www.acmicpc.net/problem/1296)

> 본인 이름과 각 팀 이름을 합쳐 L, O, V, E 글자 개수를 센 뒤, 공식으로 궁합 점수를 계산하여 가장 높은 팀을 출력한다.

## 풀이

각 팀원 이름과 본인 이름을 합쳐 L, O, V, E 개수를 세고, 조합 공식 ((L+O)*(L+V)*(L+E)*(O+V)*(O+E)*(V+E)) % 100으로 궁합 점수를 계산한다. 최대 점수인 팀을 출력하며, 동점이면 사전순으로 앞선 팀을 선택한다.

## 코드

```javascript
const solution = (input) => {
  const name = input[0].trim();
  const N = +input[1];
  const teams = [];

  for (let i = 2; i < 2 + N; i++) {
    teams.push(input[i].trim());
  }

  const count = (str, ch) => [...str].filter((c) => c === ch).length;

  let maxScore = -1;
  let result = "";

  teams.sort();

  for (const team of teams) {
    const combined = name + team;
    const L = count(combined, "L");
    const O = count(combined, "O");
    const V = count(combined, "V");
    const E = count(combined, "E");

    const score =
      ((L + O) * (L + V) * (L + E) * (O + V) * (O + E) * (V + E)) % 100;

    if (score > maxScore) {
      maxScore = score;
      result = team;
    }
  }

  return result;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N × L) (L: 이름 길이)
- 공간: O(N)
