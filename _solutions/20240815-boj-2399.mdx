## 문제

[BOJ 2399 - 거리의 합](https://www.acmicpc.net/problem/2399)

> N개의 수가 주어졌을 때, 모든 쌍 (i, j)에 대해 |a_i - a_j|의 합을 구한다.

## 풀이

이중 반복문으로 모든 (i, j) 쌍을 순회하며 `abs(map[i] - map[j])`를 누적한다. 자기 자신과의 차이도 포함하므로 (i, i) 쌍은 0이 된다. 값이 커질 수 있어 `long long` 타입을 사용한다.

## 코드

```cpp
#include <iostream>
#include <math.h>
using namespace std;
typedef long long ll;
int n;
ll map[10100];
int main()
{
  scanf("%d", &n);
  for (int i = 0; i < n; i++)
  {
    scanf("%lld", &map[i]);
  }
  ll ans = 0;
  for (int i = 0; i < n; i++)
  {
    for (int j = 0; j < n; j++)
    {
      ans += abs(map[i] - map[j]);
    }
  }
  cout << ans << endl;
}
```

## 복잡도

- 시간: O(N^2)
- 공간: O(N)
