## 문제

[BOJ 1515 - 수 이어 쓰기](https://www.acmicpc.net/problem/1515)

> 1부터 순서대로 이어 쓴 문자열에서 일부 숫자를 지운 결과가 주어질 때, 원래 이어 쓴 수 중 마지막 수를 구한다.

## 풀이

1부터 순서대로 수를 생성하며, 해당 수의 각 자릿수를 남은 문자열과 순서대로 매칭한다. 문자열의 모든 문자가 매칭되면 현재 수가 답이다. 그리디하게 매칭 가능한 숫자를 건너뛰지 않고 최대한 빨리 소진시킨다.

## 코드

```python
import sys

nums = sys.stdin.readline().rstrip()
n = 0
idx = 0
while True:
    n += 1
    for s in str(n):
        if nums[idx] == s:
            idx += 1
            if idx >= len(nums):
                print(n)
                exit()
```

## 복잡도

- 시간: O(N * d), N은 답, d는 자릿수
- 공간: O(1)
