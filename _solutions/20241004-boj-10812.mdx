## 문제

[BOJ 10812 - 바구니 순서 바꾸기](https://www.acmicpc.net/problem/10812)

> N개의 바구니가 순서대로 있을 때, M번의 회전 연산(구간 [i, j]를 k번째 위치 기준으로 회전)을 수행한 결과를 출력한다.

## 풀이

STL의 `rotate` 함수를 사용하여 구간 [i, j]를 k 위치 기준으로 왼쪽 회전시킨다. 배열 인덱스 변환에 주의하여 arr+i-1부터 arr+j까지의 구간에서 arr+k-1을 새로운 시작점으로 회전한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;
int main()
{
  int n, m, i, j, k;
  cin >> n >> m;
  int arr[100];
  for (int p = 0; p < 100; p++)
    arr[p] = p + 1;
  while (m-- > 0)
  {
    cin >> i >> j >> k;
    rotate(arr + i - 1, arr + k - 1, arr + j);
  }
  for (int p = 0; p < n; p++)
    cout << arr[p] << " ";
}
```

## 복잡도

- 시간: O(M * N) (M: 연산 횟수, N: 바구니 수)
- 공간: O(N)
