## 문제

[BOJ 16926 - 배열 돌리기 1](https://www.acmicpc.net/problem/16926)

> N x M 배열을 반시계 방향으로 R번 회전시킨 결과를 출력한다.

## 풀이

배열을 바깥쪽 테두리부터 안쪽 테두리까지 각각 독립적인 고리(ring)로 분리한다. 각 고리의 원소를 deque에 넣고, `rotate(-R)`로 반시계 방향 회전을 수행한 뒤 결과 배열에 다시 채워 넣는다. 고리의 수는 min(N, M) // 2이다.

## 코드

```python
from sys import stdin
from collections import deque

N, M, R = map(int, stdin.readline().split())

matrix = []
ans = [[0] * M for _ in range(N)]
deq = deque()

for i in range(N):
    matrix.append(list(stdin.readline().split()))

loops = min(N, M) // 2
for i in range(loops):
    deq.clear()
    deq.extend(matrix[i][i : M - i])
    deq.extend([row[M - i - 1] for row in matrix[i + 1 : N - i - 1]])
    deq.extend(matrix[N - i - 1][i : M - i][::-1])
    deq.extend([row[i] for row in matrix[i + 1 : N - i - 1]][::-1])

    deq.rotate(-R)

    for j in range(i, M - i):
        ans[i][j] = deq.popleft()
    for j in range(i + 1, N - i - 1):
        ans[j][M - i - 1] = deq.popleft()
    for j in range(M - i - 1, i - 1, -1):
        ans[N - i - 1][j] = deq.popleft()
    for j in range(N - i - 2, i, -1):
        ans[j][i] = deq.popleft()

for line in ans:
    print(" ".join(line))
```

## 복잡도

- 시간: O(N * M)
- 공간: O(N * M)
