## 문제

[BOJ 2057 - 팩토리얼 분해](https://www.acmicpc.net/problem/2057)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```python
n = int(input())

fact = [1, 1]
for i in range(2, 21):
    fact.append(fact[i-1]*i)

sum = 0

for i in range(20, -1, -1):
    if sum+fact[i] < n:
        sum += fact[i]
    elif sum+fact[i] == n:  
        print("YES")
        exit(0)

print("NO")
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
