## 문제

[BOJ 8725 - Szachy](https://www.acmicpc.net/problem/8725)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(0);
  cout.tie(0);

  int n, a, S = 0;

  cin >> n;

  for (int i = 0; i < n; ++i)
  {
    int Max = 0;

    for (int j = 0; j < n; ++j)
      cin >> a, Max = max(Max, a);

    S += Max;
  }

  cout << S;

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
