## 문제

[BOJ 2840 - 행운의 바퀴](https://www.acmicpc.net/problem/2840)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
n, k = map(int, input().split())
circle = ["?"] * n

for i in range(k):
    spin = input().split()
    s = int(spin[0]) % n
    s_char = str(spin[1])
    circle = circle[-s:] + circle[:-s]
    if circle[0] == "?":
        if s_char in circle:
            print("!")
            break
        circle[0] = s_char
    elif circle[0] == s_char:
        continue
    else:
        print("!")
        break
else:
    print("".join(circle))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
