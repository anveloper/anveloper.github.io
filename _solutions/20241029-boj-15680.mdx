## 문제

[BOJ 15680 - 연세대학교](https://www.acmicpc.net/problem/15680)

> 0이면 "Leading the Way to the Future", 1이면 "YONSEI"를 출력한다.

## 풀이

입력값이 1이면 "Leading the Way to the Future"를, 그 외이면 "YONSEI"를 출력하는 단순 조건문이다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int N;
  cin >> N;
  if (N == 1)
    cout << "Leading the Way to the Future" << endl;
  else
    cout << "YONSEI" << endl;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
