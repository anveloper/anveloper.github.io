## 문제

[BOJ 2408 - 큰 수 계산](https://www.acmicpc.net/problem/2408)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
if __name__ == "__main__":
    equation = ""
    n = int(input())
    for _ in range(n + n - 1):
        equation += input()
    equation = equation.replace("/", "//")
    print(eval(equation))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
