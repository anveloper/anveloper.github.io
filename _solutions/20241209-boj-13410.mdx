## 문제

[BOJ 13410 - 거꾸로 구구단](https://www.acmicpc.net/problem/13410)

> N × 1부터 N × K까지의 결과를 각각 뒤집었을 때 가장 큰 값을 출력한다.

## 풀이

1부터 K까지 반복하면서 N × i를 문자열로 변환 후 뒤집고, 정수로 변환한 값 중 최댓값을 구한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

int n, k;
int ans = 0;

int main()
{
  cin >> n >> k;

  for (int i = 1; i <= k; i++)
  {
    int res = n * i;
    string str = to_string(res);
    reverse(str.begin(), str.end());
    ans = max(ans, stoi(str));
  }

  cout << ans;
}
```

## 복잡도

- 시간: O(K × D) (D: 자릿수)
- 공간: O(D)
