## 문제

[BOJ 5426 - 비밀 편지](https://www.acmicpc.net/problem/5426)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
T = int(input())

for t in range(0, T):
    msg = input()
    n = int(len(msg) ** (1 / 2))

    decoded_msg = ""
    for i in range(n, 0, -1):
        for j in range(i, n * n + 1, n):
            decoded_msg += msg[j - 1]

    print(decoded_msg)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
