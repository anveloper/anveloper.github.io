## 문제

[BOJ 5365 - Decoder](https://www.acmicpc.net/problem/5365)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  cin >> n;
  size_t len = 1;
  for (int i = 0; i < n; i++)
  {
    string word;
    cin >> word;
    char decodedC;
    if (word.size() < len)
      decodedC = ' ';
    else
      decodedC = word[len - 1];
    len = word.size();

    cout << decodedC;
  }
  cout << "\n";

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
