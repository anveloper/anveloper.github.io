## 문제

[BOJ 5623 - 수열의 합](https://www.acmicpc.net/problem/5623)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <cstdio>
int S[1001][1001] = {};
int main(void)
{
  int N, one;
  scanf("%d", &N);
  int ans[1001] = {};
  for (int i = 1; i <= N; i++)
  {
    for (int j = 1; j <= N; j++)
    {
      scanf("%d", &S[i][j]);
    }
  }
  if (N != 2)
  {
    one = (S[1][2] + S[1][3] - S[2][3]) / 2;
    for (int i = 1; i <= N; i++)
    {
      if (i == 1)
        ans[i] = one;
      else
      {
        ans[i] = S[1][i] - one;
      }
      printf("%d ", ans[i]);
    }
  }
  else
  {
    printf("1 1");
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
