## 문제

[BOJ 13223 - 소금 폭탄](https://www.acmicpc.net/problem/13223)

> 현재 시각과 폭탄이 터지는 시각이 주어질 때, 남은 시간을 HH:MM:SS 형식으로 출력한다.

## 풀이

두 시각을 초 단위로 변환한 뒤 차이를 구한다. 목표 시각이 현재보다 같거나 이르면 24시간(86400초)을 더해 다음 날로 계산한다. 결과를 다시 시, 분, 초로 변환하여 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int st[3], et[3], sTotal, eTotal, ans;

int main()
{
  scanf("%d:%d:%d", st, st + 1, st + 2);
  scanf("%d:%d:%d", et, et + 1, et + 2);
  sTotal = st[0] * 3600 + st[1] * 60 + st[2];
  eTotal = et[0] * 3600 + et[1] * 60 + et[2];
  if (sTotal >= eTotal)
    eTotal += 24 * 3600;
  ans = eTotal - sTotal;
  printf("%02d:%02d:%02d", ans / 3600, (ans % 3600) / 60, ans % 60);
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
