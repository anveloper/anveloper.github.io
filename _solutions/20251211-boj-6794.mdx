## 문제

[BOJ 6794 - What is n, Daddy?](https://www.acmicpc.net/problem/6794)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;
bool check[6];
int n, cnt;
int main()
{
  cin >> n;
  for (int i = 0; i <= 5; i++)
    for (int j = 0; j <= 5; j++)
      if (i + j == n)
        check[i] = true;
  for (int i = 0; i <= n / 2; i++)
    if (check[i])
      cnt++;
  cout << cnt;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
