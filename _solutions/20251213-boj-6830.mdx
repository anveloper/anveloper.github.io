## 문제

[BOJ 6830 - It’s Cold Here!](https://www.acmicpc.net/problem/6830)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);

  int tmp, min = 300;
  string s, coldest;

  while (cin >> s >> tmp)
  {
    if (tmp <= min)
    {
      min = tmp;
      coldest = s;
    }
  }

  cout << coldest;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
