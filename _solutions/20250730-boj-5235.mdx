## 문제

[BOJ 5235 - Even Sum More Than Odd Sum](https://www.acmicpc.net/problem/5235)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int cntCase;
  cin >> cntCase;

  for (int t = 0; t < cntCase; t++)
  {
    int n;
    cin >> n;

    int sumEven = 0, sumOdd = 0;

    for (int i = 0; i < n; i++)
    {
      int num;
      cin >> num;

      if (num % 2 == 0)
        sumEven += num;
      else
        sumOdd += num;
    }

    if (sumEven > sumOdd)
      cout << "EVEN\n";
    else if (sumEven < sumOdd)
      cout << "ODD\n";
    else
      cout << "TIE\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
