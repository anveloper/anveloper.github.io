## 문제

[BOJ 1421 - 나무꾼 이다솜](https://www.acmicpc.net/problem/1421)

> N개의 나무를 같은 길이 L로 잘라 팔 때, 자르는 비용 C를 고려하여 최대 수익을 구한다.

## 풀이

가능한 모든 길이 L(1부터 최대 나무 길이까지)에 대해 각 나무의 수익을 계산한다. 나무를 L 길이로 자르면 (tree // L)개의 토막이 나오고, 자르는 횟수는 (ceil(tree / L) - 1)이다. 각 토막의 판매 수익에서 자르는 비용을 빼서 0 이상인 경우만 합산한다.

## 코드

```python
from math import ceil

n, c, w = map(int, input().split())
trees = [int(input()) for _ in range(n)]
profits = []

for L in range(1, max(trees) + 1):
    profit = []
    for tree in trees:
        profit.append(max(0, (tree // L) * L * w - (ceil(tree / L) - 1) * c))
    profits.append(sum(profit))

print(max(profits))
```

## 복잡도

- 시간: O(M * N) (M은 최대 나무 길이, N은 나무 개수)
- 공간: O(N)
