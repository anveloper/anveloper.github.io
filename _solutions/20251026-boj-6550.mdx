## 문제

[BOJ 6550 - 부분 문자열](https://www.acmicpc.net/problem/6550)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(0);
  cout.tie(0);

  string a, b;
  while (cin >> a >> b)
  {
    int aIdx = 0;
    bool isTrue = false;
    for (int i = 0; i < b.length(); i++)
    {
      if (a[aIdx] == b[i])
      {
        aIdx++;
      }
      if (aIdx == a.length())
      {
        isTrue = true;
        break;
      }
    }
    if (isTrue)
    {
      cout << "Yes\n";
    }
    else
    {
      cout << "No\n";
    }
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
