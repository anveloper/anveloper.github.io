## 문제

[BOJ 5346 - Frodo Sequence](https://www.acmicpc.net/problem/5346)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(void)
{
  cin.tie(0);
  ios::sync_with_stdio(0);

  long long int x;
  while (1)
  {
    cin >> x;
    if (!x)
      break;
    cout << (x + 1) / 2 << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
