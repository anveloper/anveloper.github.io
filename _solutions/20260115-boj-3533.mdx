## 문제

[BOJ 3533 - Explicit Formula](https://www.acmicpc.net/problem/3533)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int z = 0;
  for (int i = 0; i < 10; ++i)
  {
    int x;
    cin >> x;
    if (!cin)
      return 0;
    if (x == 0)
      ++z;
  }
  cout << (z % 4 == 2 ? 0 : 1) << '\n';
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
