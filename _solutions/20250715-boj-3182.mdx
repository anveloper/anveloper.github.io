## 문제

[BOJ 3182 - 한동이는 공부가 하기 싫어!](https://www.acmicpc.net/problem/3182)

## 풀이

그래프 탐색(BFS/DFS)을 활용하여 탐색한다.

## 코드

```python
import sys

sys.setrecursionlimit(2000)

n = int(input())
next_senior = [0] + [int(input()) for _ in range(n)]
visited_count = [0] * (n + 1)


def dfs(start, visited):
    count = 0
    current = start
    while not visited[current]:
        visited[current] = True
        count += 1
        current = next_senior[current]
    return count


max_count = -1
answer = 0

for i in range(1, n + 1):
    visited = [False] * (n + 1)
    count = dfs(i, visited)
    visited_count[i] = count
    if count > max_count:
        max_count = count
        answer = i

print(answer)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
