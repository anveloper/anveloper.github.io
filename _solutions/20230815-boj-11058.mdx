## 문제

[BOJ 11058 - 크리보드](https://www.acmicpc.net/problem/11058)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```java
package day599;

import java.io.*;

public class Day555BOJ11058크리보드 {
  static long[] dp;

  public static void main(String[] args) throws Exception {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    int n = Integer.parseInt(br.readLine());
    dp = new long[n + 1];
    System.out.println(solve(n));
  }

  static long solve(int idx) {
    if (idx <= 0)
      return 0;
    if (idx == 1)
      return dp[idx] = idx;
    if (dp[idx] > 0)
      return dp[idx];

    dp[idx] = solve(idx - 1) + 1;
    for (int i = 2; i < 5; i++)
      dp[idx] = Math.max(dp[idx], solve(idx - (i + 1)) * i);

    return dp[idx];
  }
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
