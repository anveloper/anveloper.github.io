## 문제

[BOJ 2847 - 게임을 만든 동준이](https://www.acmicpc.net/problem/2847)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```python
N = int(input())
score = []
cnt = 0

for _ in range(N):
    score.append(int(input()))

for i in range(N - 1, 0, -1):
    if score[i] <= score[i - 1]:
        cnt += score[i - 1] - score[i] + 1
        score[i - 1] = score[i] - 1

print(cnt)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
