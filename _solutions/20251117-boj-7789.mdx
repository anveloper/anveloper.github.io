## 문제

[BOJ 7789 - 텔레프라임](https://www.acmicpc.net/problem/7789)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

bool isPrime(int n)
{
  if (n < 2)
    return false;
  for (int i = 2; i * i <= n; i++)
  {
    if (n % i == 0)
      return false;
  }
  return true;
}

int main()
{
  int n, m;
  cin >> n >> m;
  cout << ((isPrime(n) && isPrime(n + m * 1000000)) ? "Yes" : "No");
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
