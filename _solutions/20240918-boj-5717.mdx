## 문제

[BOJ 5717 - 상근이의 친구들](https://www.acmicpc.net/problem/5717)

> 남자 친구 수 M과 여자 친구 수 F가 주어질 때, 친구의 총 수를 출력한다. 0 0이 입력되면 종료한다.

## 풀이

두 수를 입력받아 합을 출력하는 반복문을 둘 다 0일 때까지 수행한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int m, f;
  while (1)
  {
    cin >> m >> f;
    if (m == 0 && f == 0)
      break;
    cout << m + f << '\n';
  }
}
```

## 복잡도

- 시간: O(T) (T: 입력 횟수)
- 공간: O(1)
