## 문제

[BOJ 2635 - 수 이어가기](https://www.acmicpc.net/problem/2635)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n = int(input())

max_list = []
for i in range(1, n + 1):
    my_list = [n]
    my_list.append(i)

    idx = 1
    while True:
        next_num = my_list[idx - 1] - my_list[idx]
        if next_num < 0:
            break
        my_list.append(next_num)
        idx += 1

    if len(max_list) < len(my_list):
        max_list = my_list

print(len(max_list))
for i in max_list:
    print(i, end=' ')
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
