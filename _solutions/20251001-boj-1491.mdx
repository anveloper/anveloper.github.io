## 문제

[BOJ 1491 - 나선](https://www.acmicpc.net/problem/1491)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int N, M, x;
  cin >> N >> M;

  if (N >= M)
  {
    if (M % 2 == 0)
      x = M / 2 - 1;
    else
      x = N - M / 2 - 1;
    cout << x << " " << M / 2 << endl;
  }
  else
  {
    if (N % 2 == 0)
      x = N / 2;
    else
      x = M - N / 2 - 1;
    cout << (N - 1) / 2 << " " << x << endl;
  }
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
