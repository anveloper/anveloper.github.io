## 문제

[BOJ 2526 - 싸이클](https://www.acmicpc.net/problem/2526)

> N에 N을 곱하고 P로 나눈 나머지를 반복할 때 나타나는 싸이클의 길이를 구한다.

## 풀이

N*N mod P를 반복하면서 각 값이 처음 나타난 순서를 기록한다. 이미 나타난 값이 다시 등장하면 현재 순서와의 차이가 싸이클 길이가 된다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, p, ne, cnt = 1;
vector<int> nums(1001);

int main()
{
  cin >> n >> p;
  ne = n;
  while (!nums[ne])
  {
    nums[ne] = cnt++;
    ne = ne * n % p;
  }
  cout << cnt - nums[ne] << '\n';
}
```

## 복잡도

- 시간: O(P)
- 공간: O(P)
