## 문제

[BOJ 4328 - 기초 나머지 계산](https://www.acmicpc.net/problem/4328)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
while 1:
    li = list(map(int, input().split()))
    if li[0] == 0:
        break
    b, p, m = li[0], li[1], li[2]
    n = int(str(p), b) % int(str(m), b)
    res = []
    while n >= b:
        res.append(str(n % b))
        n = n // b
    res.append(str(n))
    print(int("".join(res[::-1])))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
