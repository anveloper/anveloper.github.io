## 문제

[BOJ 2018 - 수들의 합 5](https://www.acmicpc.net/problem/2018)

## 풀이

투 포인터 기법을 활용한다.

## 코드

```python
n = int(input())

result = 1
for i in range(1,(n//2)+1):
  hap = i
  for j in range(i+1,n):
    hap += j
    if hap == n:
      result += 1
      break
    elif hap > n:
      break
    
print(result)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
