## 문제

[BOJ 7891 - Can you add this?](https://www.acmicpc.net/problem/7891)

> T개의 테스트 케이스에 대해 두 정수의 합을 출력한다.

## 풀이

테스트 케이스 수 T를 입력받고, 각 케이스마다 두 수를 입력받아 합을 출력한다.

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;
int t;
int main()
{
  cin >> t;
  while (t--)
  {
    int a, b;
    cin >> a >> b;
    cout << a + b << '\n';
  }
}
```

## 복잡도

- 시간: O(T)
- 공간: O(1)
