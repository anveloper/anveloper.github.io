## 문제

[BOJ 6207 - Cow Picnic](https://www.acmicpc.net/problem/6207)

## 풀이

그래프 탐색(BFS/DFS)을 활용하여 탐색한다.

## 코드

```cpp
#include <iostream>
#include <vector>
#include <cstring>

using namespace std;

typedef long long ll;

vector<int> graph[1005];
int cnt[1005];
bool v[1005];

void dfs(int cur)
{
  v[cur] = 1;
  cnt[cur]++;
  for (int nxt : graph[cur])
  {
    if (!v[nxt])
      dfs(nxt);
  }
}

int main()
{
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  int k, n, m;
  cin >> k >> n >> m;
  vector<int> arr(k);
  for (int i = 0; i < k; i++)
    cin >> arr[i];
  while (m--)
  {
    int a, b;
    cin >> a >> b;
    graph[a].emplace_back(b);
  }
  for (int x : arr)
  {
    memset(v, 0, sizeof(v));
    dfs(x);
  }
  int ans = 0;
  for (int i = 1; i <= n; i++)
    if (cnt[i] == k)
      ans++;
  cout << ans;
}
```

## 복잡도

- 시간: O(V + E)
- 공간: O(V + E)
