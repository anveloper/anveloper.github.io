## 문제

[BOJ 11942 - 고려대는 사랑입니다](https://www.acmicpc.net/problem/11942)

"고려대학교"를 출력하라.

### 입력

입력은 없다.

### 출력

"고려대학교"를 출력한다.

### 예제

| 입력 | 출력 |
| --- | --- |
| (없음) | `고려대학교` |

## 풀이

고정 문자열을 그대로 출력한다.

1. "고려대학교"를 `cout`으로 출력한다

**핵심 아이디어**: 입력 없이 고정 문자열만 출력하는 기본 출력 문제이다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  cout << "고려대학교" << endl;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
