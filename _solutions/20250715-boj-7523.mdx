## 문제

[BOJ 7523 - Gauß](https://www.acmicpc.net/problem/7523)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
for i in range(int(input())):
    a, b = map(int, input().split())
    res = (b - a + 1) * (a + b) // 2
    print(f"Scenario #{i+1}:")
    print(f"{res}\n")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
