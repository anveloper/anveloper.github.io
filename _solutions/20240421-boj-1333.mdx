## 문제

[BOJ 1333 - 부재중 전화](https://www.acmicpc.net/problem/1333)

> 노래 N곡을 연속 재생하고 곡 사이 5초 쉬는 동안, D초 간격으로 전화벨이 울린다. 쉬는 시간에 벨이 울리는 최초 시각을 구한다.

## 풀이

노래 재생/쉬는 구간을 타임라인으로 표현하고, 전화벨 간격 D씩 증가하며 쉬는 구간에 해당하는 최초 시각을 찾는다.

## 코드

```javascript
const solution = (input) => {
  const [N, L, D] = input[0].trim().split(" ").map(Number);

  const totalTime = N * L + (N - 1) * 5;

  const isPlaying = (t) => {
    if (t >= totalTime) return false;
    const cycle = L + 5;
    const pos = t % cycle;
    return pos < L;
  };

  for (let t = 0; ; t += D) {
    if (!isPlaying(t)) return t;
  }
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N × L / D)
- 공간: O(1)
