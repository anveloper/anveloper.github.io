## 문제

[BOJ 3231 - 카드놀이](https://www.acmicpc.net/problem/3231)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline

n = int(input())

arr = [0] * 100001
for i in range(n):
    num = int(input())
    arr[num] = i + 1

ans = 0
for i in range(1, n):
    if arr[i] > arr[i + 1]:
        ans += 1

print(ans)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
