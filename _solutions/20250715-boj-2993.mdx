## 문제

[BOJ 2993 - 세 부분](https://www.acmicpc.net/problem/2993)

## 풀이

정렬을 활용하여 문제를 해결한다.

## 코드

```python
s = input()
result = s
for i in range(1, len(s)):
    for j in range(i + 1, len(s)):
        result = min(result, s[:i][::-1] + s[i:j][::-1] + s[j:][::-1])
print(result)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
