## 문제

[BOJ 5217 - 쌍의 합](https://www.acmicpc.net/problem/5217)

> 1 이상 12 이하의 두 수의 합이 N이 되는 쌍을 모두 출력한다 (i가 j보다 작은 경우만).

## 풀이

1부터 12까지의 이중 루프를 돌면서 두 수의 합이 N이고, 첫 번째 수가 두 번째 수보다 작은 경우에만 출력한다. 첫 번째 쌍과 이후 쌍의 구분자로 쉼표를 사용한다.

## 코드

```cpp
#include <iostream>
#define ll long long
using namespace std;
int main()
{
  int t;
  cin >> t;
  while (t--)
  {
    int n;
    int cnt = 0;
    cin >> n;
    cout << "Pairs for " << n << ": ";
    for (int i = 1; i <= 12; i++)
      for (int j = 1; j <= 12; j++)
        if (i + j == n && i < j)
        {
          if (!cnt)
          {
            cnt++;
            cout << i << ' ' << j;
          }
          else
            cout << ", " << i << ' ' << j;
        }
    cout << '\n';
  }
}
```

## 복잡도

- 시간: O(T * 144)
- 공간: O(1)
