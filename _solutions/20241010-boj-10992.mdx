## 문제

[BOJ 10992 - 별 찍기 - 17](https://www.acmicpc.net/problem/10992)

> 삼각형의 외곽만 별로 출력하되, 마지막 줄은 별을 2*N-1개 모두 출력한다.

## 풀이

마지막 줄은 2*N-1개의 별을 모두 출력하고, 나머지 줄은 삼각형 테두리만 출력한다. 첫 줄은 별 하나, 중간 줄은 양 끝에만 별을 배치하고 나머지는 공백으로 채운다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  int num;
  cin >> num;
  for (int i = 1; i <= num; i++)
  {
    if (i == num)
    {
      for (int i = 1; i <= 2 * num - 1; i++)
        cout << "*";
    }
    else
    {
      for (int j = num - i; j > 0; j--)
      {
        cout << " ";
      }
      cout << "*";
      if (i != 1)
      {
        for (int j = 1; j <= (i - 1) * 2 - 1; j++)
        {
          cout << " ";
        }
        cout << "*";
      }
    }
    cout << endl;
  }
  return 0;
}
```

## 복잡도

- 시간: O(N^2)
- 공간: O(1)
