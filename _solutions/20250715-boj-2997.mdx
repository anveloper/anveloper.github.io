## 문제

[BOJ 2997 - 네 번째 수](https://www.acmicpc.net/problem/2997)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
a, b, c = sorted(list(map(int, input().split())))

n1, n2 = b - a, c - b
if n1 == n2:
    print(c + n1)
else:
    n = min(n1, n2)
    if n2 != n:
        print(c - n)
    else:
        print(b - n)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
