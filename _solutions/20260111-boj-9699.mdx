## 문제

[BOJ 9699 - RICE SACK](https://www.acmicpc.net/problem/9699)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int n, max, a;
  cin >> n;
  for (int i = 0; i < n; i++)
  {
    max = 0;
    for (int j = 0; j < 5; j++)
    {
      cin >> a;
      max = max > a ? max : a;
    }
    cout << "Case #" << i + 1 << ": " << max << "\n";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
