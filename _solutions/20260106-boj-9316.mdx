## 문제

[BOJ 9316 - Hello Judge](https://www.acmicpc.net/problem/9316)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main(void)
{
  int N;

  cin >> N;

  for (int i = 1; i <= N; i++)
  {
    printf("Hello World, Judge %d!\n", i);
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
