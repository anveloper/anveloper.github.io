## 문제

[BOJ 1239 - 차트](https://www.acmicpc.net/problem/1239)

> N개의 백분율 데이터로 원형 차트를 그릴 때, 원의 지름 위에 놓이는 경계선의 최대 개수를 구한다.

## 풀이

N이 작으므로(최대 8) 전체 순열을 탐색한다. 각 순열에서 누적 합 배열을 구하고, 합이 정확히 50인 쌍의 개수를 세어 최대값을 갱신한다.

## 코드

```javascript
const solution = (input) => {
  const N = +input[0],
    list = input[1].split(" ").map((e) => +e);

  let res = 0;
  const getNumber = (arr) => {
    let a = 0;
    for (let i = 0; i < N; i++) {
      if (arr.includes(arr[i] + 50)) a++;
    }
    return a;
  };

  const getAcc = (arr) => {
    const res = [list[arr[0]]];
    for (let i = 1; i < N; i++) {
      res[i] = res[i - 1] + list[arr[i]];
    }
    return res;
  };

  const recur = (arr = []) => {
    if (arr.length === N) {
      return (res = Math.max(res, getNumber(getAcc(arr))));
    }
    for (let i = 0; i < N; i++) {
      if (arr.includes(i)) continue;
      arr.push(i);
      recur(arr);
      arr.pop();
    }
  };
  recur();
  return res;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(N! × N)
- 공간: O(N)
