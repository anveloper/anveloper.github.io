## 문제

[BOJ 1531 - 투명](https://www.acmicpc.net/problem/1531)

> 100×100 그림 위에 N장의 불투명 종이를 올렸을 때, M장 초과로 덮인 칸의 수를 구한다.

## 풀이

100×100 2차원 배열에 각 종이의 영역을 1씩 증가시킨 뒤, M 초과인 칸의 수를 세면 된다.

## 코드

```python
N, M = map(int, input().split())
li = [[0]*100 for _ in range(100)]
for _ in range(N):
    x1, y1, x2, y2 = map(int, input().split())
    for i in range(x1, x2+1):
        for j in range(y1, y2+1):
            li[i-1][j-1] += 1
cnt = 0
for i in range(100):
    for j in range(100):
        if li[i][j] > M:
            cnt += 1
print(cnt)
```

## 복잡도

- 시간: O(N × 면적 + 10000)
- 공간: O(10000)
