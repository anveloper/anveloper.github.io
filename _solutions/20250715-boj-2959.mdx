## 문제

[BOJ 2959 - 거북이](https://www.acmicpc.net/problem/2959)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
li = sorted(map(int, input().split()))
print(li[0] * li[2])
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
