## 문제

[BOJ 5766 - 할아버지는 유명해!](https://www.acmicpc.net/problem/5766)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline

while True:
    N, M = map(int, input().split())
    if not N and not M:
        break
    ranking = [0] * 10001
    for _ in range(N):
        for i in list(map(int, input().split())):
            ranking[i] += 1

    total = sorted(set(ranking), reverse=True)
    second = total[1]
    for i, j in enumerate(ranking):
        if j == second:
            print(i, end=" ")

    print()
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
