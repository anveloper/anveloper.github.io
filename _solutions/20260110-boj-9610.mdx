## 문제

[BOJ 9610 - 사분면](https://www.acmicpc.net/problem/9610)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  cin.tie(NULL);
  ios::sync_with_stdio(false);

  int n;
  cin >> n;

  int ans[5] = {};

  int x, y;

  for (int i = 0; i < n; i++)
  {
    cin >> x >> y;

    if (x > 0 && y > 0)
      ans[0]++;
    else if (x < 0 && y > 0)
      ans[1]++;
    else if (x < 0 && y < 0)
      ans[2]++;
    else if (x > 0 && y < 0)
      ans[3]++;
    else
      ans[4]++;
  }
  for (int i = 0; i < 4; i++)
  {
    cout << "Q" << i + 1 << ": " << ans[i] << "\n";
  }
  cout << "AXIS: " << ans[4] << "\n";
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
