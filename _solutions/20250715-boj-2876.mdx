## 문제

[BOJ 2876 - 그래픽스 퀴즈](https://www.acmicpc.net/problem/2876)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```python
input = open(0).readline


def solve():
    N = int(input())
    cnt = [0] * 5
    res = (0, 5)
    for _ in range(N):
        A, B = map(int, input().split())
        for i in range(1, 6):
            if i in (A, B):
                cnt[i - 1] += 1
                if res[0] < cnt[i - 1] or (res[0] == cnt[i - 1] and res[1] > i):
                    res = (cnt[i - 1], i)
            else:
                cnt[i - 1] = 0
    print(*res)


solve()
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
