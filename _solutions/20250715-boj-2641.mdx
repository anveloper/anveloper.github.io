## 문제

[BOJ 2641 - 다각형그리기](https://www.acmicpc.net/problem/2641)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline

n = int(input())
s1 = input().strip().replace(" ", "")
s1 *= 2
table = str.maketrans("1234", "3412")
s2 = s1[::-1].translate(table)
answer = []
for _ in range(int(input())):
    x = input().strip()
    y = x.strip().replace(" ", "")
    if y in s1 or y in s2:
        answer.append(x)
print(len(answer))
print(*answer, sep="\n")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
