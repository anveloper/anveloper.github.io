## 문제

[BOJ 6750 - Rotating letters](https://www.acmicpc.net/problem/6750)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

char letter[7] = {'I', 'O', 'S', 'H', 'Z', 'X', 'N'};

bool canUse(char c);

bool canUse(char c)
{
  for (int i = 0; i < 7; i++)
  {
    if (c == letter[i])
      return true;
  }

  return false;
}

int main()
{
  string s;
  cin >> s;

  for (int i = 0; i < s.length(); i++)
  {
    if (!canUse(s[i]))
    {
      cout << "NO\n";
      return 0;
    }
  }

  cout << "YES\n";

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
