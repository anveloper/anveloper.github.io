## 문제

[BOJ 8741 - 이진수 합](https://www.acmicpc.net/problem/8741)

> 정수 N이 주어질 때, 1을 N개, 0을 N-1개 이어붙인 수(111...000...)를 출력한다.

## 풀이

먼저 1을 N번 출력하고, 이어서 0을 N-1번 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(void)
{
  int N;
  cin >> N;

  for (int i = 0; i < N; i++)
  {
    cout << 1;
  }
  for (int i = 0; i < N - 1; i++)
  {
    cout << 0;
  }
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
