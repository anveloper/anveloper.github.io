## 문제

[BOJ 5357 - Dedupe](https://www.acmicpc.net/problem/5357)

> 문자열에서 연속으로 같은 문자가 반복되면 하나만 남기고 제거한 결과를 출력한다.

## 풀이

이전 문자를 기록하면서 현재 문자가 이전과 다를 때만 출력하는 방식으로 연속 중복을 제거한다.

## 코드

```cpp
#include <iostream>
#include <cstring>
#include <algorithm>
#include <vector>
#include <cmath>

using namespace std;

#define fast                        \
  ios_base::sync_with_stdio(false); \
  cin.tie(0), cout.tie(0)
#define ll long long

int main()
{
  fast;
  int n;
  string s;
  cin >> n;
  while (n--)
  {
    char prev = ' ';
    cin >> s;
    for (int i = 0; i < s.length(); i++)
    {
      if (prev != s[i])
      {
        cout << s[i];
        prev = s[i];
      }
    }
    cout << "\n";
  }
}
```

## 복잡도

- 시간: O(N × L) (L: 문자열 길이)
- 공간: O(L)
