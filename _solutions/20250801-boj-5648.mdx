## 문제

[BOJ 5648 - 역원소 정렬](https://www.acmicpc.net/problem/5648)

## 풀이

정렬을 활용하여 문제를 해결한다.

## 코드

```cpp
#include <stdio.h>
#include <algorithm>
using namespace std;
typedef long long ll;
ll a[1000002], i, n, x, y;
int main()
{
  scanf("%lld", &n);
  for (i = 0; i < n; ++i)
  {
    scanf("%lld", &x);
    y = 0;
    while (x > 0)
    {
      y = 10 * y + (x % 10);
      x /= 10;
    }
    a[i] = y;
  }
  sort(a, a + n);
  for (i = 0; i < n; ++i)
    printf("%lld\n", a[i]);
  return 0;
}
```

## 복잡도

- 시간: O(N log N)
- 공간: O(N)
