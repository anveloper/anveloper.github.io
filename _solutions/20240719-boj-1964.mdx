## 문제

[BOJ 1964 - 오각형, 오각형, 오각형…](https://www.acmicpc.net/problem/1964)

> N번째 오각형에 포함된 점의 개수를 45678로 나눈 나머지를 출력한다.

## 풀이

오각형 수의 점 개수에 대한 수학 공식을 유도하여 한 번에 계산한다. N이 1이면 5를 출력하고, 그 외에는 등차수열의 합 공식을 이용하여 결과를 45678로 나눈 나머지를 구한다.

## 코드

```cpp
#include <iostream>

int main(void)
{
  int N;
  std::cin >> N;

  if (N == 1)
    std::cout << 5;
  else
    std::cout << ((long)(((2 + N + 1) / (double)2) * N * 3 - 2 * N) + 1) % 45678;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
