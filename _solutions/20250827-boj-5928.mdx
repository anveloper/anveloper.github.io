## 문제

[BOJ 5928 - Contest Timing](https://www.acmicpc.net/problem/5928)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int d, h, m;
int main()
{
  cin >> d >> h >> m;
  int pivot = 11 + 11 * 60 + 11 * 60 * 24;
  int ans = m + h * 60 + d * 60 * 24 - pivot;
  if (ans < 0)
    cout << -1 << '\n';
  else
    cout << ans << '\n';
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
