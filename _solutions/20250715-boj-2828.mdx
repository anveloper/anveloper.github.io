## 문제

[BOJ 2828 - 사과 담기 게임](https://www.acmicpc.net/problem/2828)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```python
import sys

N, M = map(int, sys.stdin.readline().split())
J = int(sys.stdin.readline())
now = 1
apples = []
answer = 0
for _ in range(J):
    apples.append(int(sys.stdin.readline()))
for apple in apples:
    if now <= apple and now + (M - 1) >= apple:
        pass
    elif now > apple:
        answer += abs(apple - now)
        now = apple
    else:
        answer += apple - (M - 1) - now
        now = apple - (M - 1)
print(answer)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
