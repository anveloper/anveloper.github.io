## 문제

[BOJ 2089 - -2진수](https://www.acmicpc.net/problem/2089)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n = int(input())
res =''
if n == 0:
    print(0)
    exit()
while n:
    if n % (-2):
        res = '1' + res
        n = n//-2 + 1
    else:
        res = '0' + res
        n = n//-2

print(res)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
