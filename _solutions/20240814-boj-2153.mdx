## 문제

[BOJ 2153 - 소수 단어](https://www.acmicpc.net/problem/2153)

> 소문자는 1~26, 대문자는 27~52의 값을 가지며, 단어의 각 문자 값 합이 소수이면 소수 단어이다.

## 풀이

각 문자에 대해 소문자는 `a`=1부터, 대문자는 `A`=27부터 값을 매겨 합산한다. 합산된 값이 소수인지 판별하기 위해 2부터 `sqrt(count)`까지 나누어 떨어지는지 확인한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

int main()
{
  string word;
  cin >> word;

  int count = 0;
  int len = word.length();
  for (int i = 0; i < len; i++)
  {
    if (word[i] >= 'a' && word[i] <= 'z')
    {
      count += (int)(word[i] - 'a' + 1);
    }
    else if (word[i] >= 'A' && word[i] <= 'Z')
    {
      count += (int)(word[i] - 'A' + 27);
    }
  }

  bool flag = 0;
  for (int i = 2; i * i <= count; i++)
  {
    if (count % i == 0)
    {
      flag = 1;
      break;
    }
  }

  if (flag)
  {
    cout << "It is not a prime word.";
  }
  else
  {
    cout << "It is a prime word.";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N + sqrt(S)) (N: 문자열 길이, S: 문자 값 합)
- 공간: O(1)
