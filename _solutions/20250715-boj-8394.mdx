## 문제

[BOJ 8394 - 악수](https://www.acmicpc.net/problem/8394)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```python
x = int(input())

a, b = 1, 2
if x == 1:
    print(a)
elif x == 2:
    print(b)
else:
    for i in range(x - 2):
        a, b = b, (a + b) % 10
    print(b)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
