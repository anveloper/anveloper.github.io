## 문제

[BOJ 2531 - 회전 초밥](https://www.acmicpc.net/problem/2531)

## 풀이

투 포인터 기법을 활용한다.

## 코드

```python
from collections import defaultdict
import sys
input = sys.stdin.readline

n, d, k, c = map(int, input().split())
belt = [int(input()) for _ in range(n)]
count = defaultdict(int) 

for i in range(k):
    count[belt[i]] += 1
    
count[c] += 1  
total = len(count)   

for i in range(1, n):
    start = belt[i-1]
    count[start] -= 1
    
    if not count[start]:
        count.pop(start)

    end = belt[(i + k - 1) % n]
    count[end] += 1

    total = max(total, len(count))

print(total)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
