## 문제

[BOJ 3054 - 피터팬 프레임](https://www.acmicpc.net/problem/3054)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
w = input()
result = [".", ".", "#", ".", "."]
for n, i in enumerate(w, 1):
    s = "#"
    if n % 3 == 0:
        s = "*"
        result[2] = result[2][:-1] + s
    result[0] += f".{s}.."
    result[1] += f"{s}.{s}."
    result[2] += f".{i}.{s}"
    result[3] += f"{s}.{s}."
    result[4] += f".{s}.."

for i in result:
    print(i)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
