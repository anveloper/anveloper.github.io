## 문제

[BOJ 5102 - Sarah's Toys](https://www.acmicpc.net/problem/5102)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  while (1)
  {
    int a, b, diff;
    cin >> a >> b;
    if (!a && !b)
      break;
    diff = a - b;
    if (diff < 2)
      cout << 0 << ' ' << 0;
    else if (diff % 2)
      cout << (diff - 3) / 2 << ' ' << 1;
    else
      cout << diff / 2 << ' ' << 0;
    cout << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
