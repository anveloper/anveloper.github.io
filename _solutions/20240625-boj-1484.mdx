## 문제

[BOJ 1484 - 다이어트](https://www.acmicpc.net/problem/1484)

> 현재 몸무게^2 - 기억하는 몸무게^2 = G를 만족하는 현재 몸무게를 모두 구한다.

## 풀이

a^2 - b^2 = G에서 (a+b)(a-b) = G이다. G의 약수 쌍 (d, G/d)에서 a-b = d, a+b = G/d로 놓으면, a = (d + G/d) / 2이다. d가 G의 약수이고 d와 G/d의 합이 짝수이며 d != G/d인 경우에만 유효한 해가 존재한다. sqrt(G)까지만 탐색하면 충분하다.

## 코드

```python
import sys
import math

input = sys.stdin.readline

G = int(input())

ans = []
for a in range(1, math.ceil(math.sqrt(G))):
    if G % a == 0:
        b = int(G / a)

        if a == b or (a + b) % 2 == 1:
            continue
        ans.append(int((a + b) / 2))

if not ans:
    print(-1)
else:
    ans.sort()
    for n in ans:
        print(n)
```

## 복잡도

- 시간: O(sqrt(G))
- 공간: O(sqrt(G))
