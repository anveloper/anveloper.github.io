## 문제

[BOJ 5893 - 17배](https://www.acmicpc.net/problem/5893)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
n = int(input(), 2)
n *= 17
print(bin(n)[2:])
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
