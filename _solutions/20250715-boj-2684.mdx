## 문제

[BOJ 2684 - 동전 게임](https://www.acmicpc.net/problem/2684)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
for _ in range(int(input())):
    d = {"TTT": 0, "TTH": 0, "THT": 0, "THH": 0, "HTT": 0, "HTH": 0, "HHT": 0, "HHH": 0}
    s = input()
    for i in range(38):
        d[s[i : i + 3]] += 1
    for t, n in d.items():
        print(n, end=" ")
    print()
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
