## 문제

[BOJ 2942 - 퍼거슨과 사과](https://www.acmicpc.net/problem/2942)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
a, b = map(int, input().split())


def gcd(a, b):
    while a:
        a, b = b % a, a
    return b


g = gcd(a, b)
a = a // g
b = b // g

for i in range(1, int(g**0.5) + 1):
    if g % i:
        continue
    h = g // i
    print(i, h * a, h * b)
    if h != i:
        print(h, i * a, i * b)
```

## 복잡도

- 시간: O(log N)
- 공간: O(1)
