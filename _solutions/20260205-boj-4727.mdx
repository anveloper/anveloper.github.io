## 문제

[BOJ 4727 - Close Enough Computations](https://www.acmicpc.net/problem/4727)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main(void) {
	cin.tie(0);
	ios::sync_with_stdio(0);

	while (1) {
		int n, a, b, c; cin >> n >> a >> b >> c; if (!n) break;
		int L = 0, R = 0;

		if (a) L += (9 * a - 4), R += (9 * a + 4);
		else R += 4;

		if (b) L += (4 * b - 2), R += (4 * b + 2);
		else R += 2;

		if (c) L += (4 * c - 2), R += (4 * c + 2);
		else R += 2;

		if (L <= n && n <= R) cout << "yes\n";
		else cout << "no\n";
	}
	return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
