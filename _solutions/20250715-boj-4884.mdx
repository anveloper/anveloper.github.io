## 문제

[BOJ 4884 - FIFA 월드컵](https://www.acmicpc.net/problem/4884)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys

while True:
    g, t, a, d = map(int, sys.stdin.readline().split())
    if g == -1:
        break

    pre = t * (t - 1) // 2 * g
    knockout = g * a + d

    x, y = 0, 0
    temp = 1
    while knockout > temp:
        x += temp
        temp *= 2

    y += temp - knockout
    x += pre

    print(f"{g}*{a}/{t}+{d}={x}+{y}")
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
