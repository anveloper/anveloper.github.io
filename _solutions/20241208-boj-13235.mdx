## 문제

[BOJ 13235 - 팰린드롬](https://www.acmicpc.net/problem/13235)

> 주어진 문자열이 앞뒤로 뒤집어도 같은 팰린드롬인지 판별한다.

## 풀이

입력 문자열을 뒤집은 문자열과 비교하여 같으면 true, 다르면 false를 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

string s1, s2;

int main()
{
  ios::sync_with_stdio(0);
  cin.tie(0);

  cin >> s1;
  s2 = s1;
  reverse(s2.begin(), s2.end());

  if (s1 == s2)
    cout << "true";
  else
    cout << "false";
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
