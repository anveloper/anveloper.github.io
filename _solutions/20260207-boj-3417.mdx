## 문제

[BOJ 3417 - Vigenère Cipher Encryption](https://www.acmicpc.net/problem/3417)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main() {
  string s1, s2;
  while (cin >> s1 >> s2) {
    int i(0);
    for (char c : s2) {
      cout << char((c - 65 + (s1[i] - 64)) % 26 + 65);
      i = (i + 1) % s1.length();
    }
    cout << "\n";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
