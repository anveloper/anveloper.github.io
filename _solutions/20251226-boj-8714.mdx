## 문제

[BOJ 8714 - Monety](https://www.acmicpc.net/problem/8714)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;
int n, cnt;

int main()
{
  cin >> n;
  for (int i = 0, s; i < n; i++)
  {
    cin >> s;
    if (s)
      cnt++;
  }
  cout << min(cnt, n - cnt);
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
