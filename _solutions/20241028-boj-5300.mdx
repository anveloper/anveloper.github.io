## 문제

[BOJ 5300 - Fill the Rowboats!](https://www.acmicpc.net/problem/5300)

1부터 N까지의 번호를 출력하면서, 6명씩 보트에 태울 때마다(6의 배수 또는 마지막) "Go!"를 출력하라.

### 입력

정수 N이 주어진다.

### 출력

번호를 출력하되 6의 배수이거나 마지막 번호에서 "Go!"를 추가한다.

### 예제

| 입력 | 출력                      |
| ---- | ------------------------- |
| `8`  | `1 2 3 4 5 6 Go! 7 8 Go!` |

## 풀이

1부터 N까지 순회하며 조건에 따라 "Go!"를 추가한다.

1. 각 번호를 출력한다
2. 6의 배수이거나 마지막 번호이면 "Go!"를 함께 출력한다

**핵심 아이디어**: 6명 단위로 보트를 출발시키는 시뮬레이션으로, 마지막 그룹은 6명 미만이어도 "Go!"를 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  int N;
  cin >> N;
  for (int i = 1; i <= N; i++)
  {
    if (i != N)
    {
      if (i % 6 == 0)
        cout << i << " Go! ";
      else
        cout << i << " ";
    }
    else
      cout << i << " Go! ";
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(1)
