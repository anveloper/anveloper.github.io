## 문제

[BOJ 2223 - 금화](https://www.acmicpc.net/problem/2223)

## 풀이

그리디 알고리즘으로 매 단계 최선의 선택을 한다.

## 코드

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main()
{
  int t, x, m;
  int answer = 0;
  cin >> t >> x >> m;

  int d, s;
  int minV = 10000000;
  for (int i = 0; i < m; i++)
  {
    cin >> d >> s;
    minV = min(minV, (d - 1) / s);
  }

  if (t >= minV)
  {
    answer = x * (minV + (t - minV) / 2);
  }
  if (t < minV)
  {
    answer = x * t;
  }
  if (minV == 0)
  {
    answer = 0;
  }
  if (m == 0)
  {
    answer = x * t;
  }

  cout << answer;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
