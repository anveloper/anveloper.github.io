## 문제

[BOJ 1788 - 피보나치 수의 확장](https://www.acmicpc.net/problem/1788)

## 풀이

다이나믹 프로그래밍을 사용하여 부분 문제의 해를 메모이제이션하며 풀이한다.

## 코드

```java
package day499;

import java.io.*;

public class Day462BOJ1788피보나치수의확장 {
  static final int MOD = 1_000_000_000;
  static final int SET = 1_000_000;
  static int N, ans = 1;
  static long[] dp = new long[2000001];

  public static void main(String[] args) throws Exception {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    N = Integer.parseInt(br.readLine()) + SET;

    dp[SET + 1] = 1;
    if (N < SET) {
      for (int i = SET - 1; i >= N; i--) {
        dp[i] = (dp[i + 2] - dp[i + 1]) % MOD;
      }
    } else {
      for (int i = SET + 2; i <= N; i++) {
        dp[i] = (dp[i - 1] + dp[i - 2]) % MOD;
      }
    }
    if (dp[N] == 0)
      ans = 0;
    else if (dp[N] < 0)
      ans = -1;
    System.out.println(ans);
    System.out.print(Math.abs(dp[N]));
  }
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
