## 문제

[BOJ 6030 - Scavenger Hunt](https://www.acmicpc.net/problem/6030)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

vector<int> divisors_sorted(int n)
{
  vector<int> lo, hi;
  for (int i = 1; i * i <= n; ++i)
  {
    if (n % i == 0)
    {
      lo.push_back(i);
      if (i * i != n)
        hi.push_back(n / i);
    }
  }
  reverse(hi.begin(), hi.end());
  lo.insert(lo.end(), hi.begin(), hi.end());
  return lo;
}

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int P, Q;
  if (!(cin >> P >> Q))
    return 0;

  auto A = divisors_sorted(P);
  auto B = divisors_sorted(Q);

  for (int a : A)
    for (int b : B)
      cout << a << ' ' << b << '\n';
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
