## 문제

[BOJ 8760 - Schronisko](https://www.acmicpc.net/problem/8760)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>

int main()
{
  int n, m, k;
  std::cin >> k;
  for (int i = 0; i < k; i++)
  {
    std::cin >> n >> m;
    std::cout << n * m / 2 << std::endl;
  }
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
