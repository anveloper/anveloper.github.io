## 문제

[BOJ 8558 - Silnia](https://www.acmicpc.net/problem/8558)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import math

n = int(input())
print(math.factorial(n) % 10)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
