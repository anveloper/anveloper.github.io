## 문제

[BOJ 1000 - A+B](https://www.acmicpc.net/problem/1000)

두 정수 A와 B를 입력받아 A+B를 출력하라.

### 입력

첫째 줄에 A와 B가 주어진다 (0 이상 10 이하).

### 출력

A+B를 출력한다.

### 예제

| 입력  | 출력 |
| ----- | ---- |
| `1 2` | `3`  |

## 풀이

표준 입력으로 두 정수를 받아 합을 출력한다.

1. `input().split()`으로 한 줄에서 두 값을 분리한다
2. `map(int, ...)`으로 정수 변환 후 합을 출력한다

**핵심 아이디어**: 가장 기본적인 입출력 문제로, Python의 `map`과 `split`으로 간결하게 처리한다.

## 코드

```python
a, b = map(int, input().split())
print(a + b)
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
