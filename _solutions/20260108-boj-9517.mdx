## 문제

[BOJ 9517 - 아이 러브 크로아티아](https://www.acmicpc.net/problem/9517)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  int k, q;
  int time = 0;
  cin >> k >> q;
  while (q--)
  {
    int t;
    char ans;
    cin >> t >> ans;
    time += t;
    if (time >= 210)
    {
      cout << k;
      break;
    }
    if (ans == 'T')
    {
      k++;
      if (k == 9)
        k = 1;
    }
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
