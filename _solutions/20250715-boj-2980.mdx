## 문제

[BOJ 2980 - 도로와 신호등](https://www.acmicpc.net/problem/2980)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline

N, L = map(int, input().split())
pos = 0
answer = 0

for _ in range(N):
    d, r, g = map(int, input().split())

    answer += d - pos
    pos = d
    if answer % (r + g) <= r:
        answer += r - (answer % (r + g))

answer += L - pos
print(answer)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
