## 문제

[BOJ 2154 - 수 이어 쓰기 3](https://www.acmicpc.net/problem/2154)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
sn = input()
n = int(sn)

snum = "".join(map(str, range(1, n + 1)))
pos = snum.find(sn) + 1

print(pos)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
