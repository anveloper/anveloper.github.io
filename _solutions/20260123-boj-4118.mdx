## 문제

[BOJ 4118 - Fred’s Lotto Tickets](https://www.acmicpc.net/problem/4118)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <set>

using namespace std;

set<int> s;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(0);

  int n, t;

  while (true)
  {
    cin >> n;
    if (n == 0)
    {
      break;
    }

    for (int i = 0; i < n * 6; i++)
    {
      cin >> t;
      s.insert(t);
    }

    if (s.size() == 49)
    {
      cout << "Yes\n";
    }
    else
    {
      cout << "No\n";
    }
    s.clear();
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
