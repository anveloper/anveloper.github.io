## 문제

[BOJ 7581 - Cuboids](https://www.acmicpc.net/problem/7581)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <vector>
#include <iostream>

using namespace std;

int main()
{
  while (1)
  {
    int zeroIdx, val = 1;
    vector<int> len(4);

    for (int i = 0; i < 4; i++)
    {
      cin >> len[i];
      if (!len[i])
        zeroIdx = i;
    }

    if (!len[0] && !len[1] && !len[2] && !len[3])
      break;

    for (int i = 0; i < 3; i++)
      if (len[i])
        val *= len[i];

    if (len[3])
      val = len[3] / val;

    for (int i = 0; i < 4; i++)
    {
      if (i == zeroIdx)
        cout << val << ' ';
      else
        cout << len[i] << ' ';
    }
    cout << '\n';
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
