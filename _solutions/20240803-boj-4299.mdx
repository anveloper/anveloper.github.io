## 문제

[BOJ 4299 - AFC 윔블던](https://www.acmicpc.net/problem/4299)

> 두 팀의 점수 합 A와 점수 차 B가 주어질 때 각 팀의 점수를 구한다. 불가능하면 -1을 출력한다.

## 풀이

x = (A+B)/2, y = (A-B)/2로 구한 뒤 x+y = A, x-y = B가 성립하는지 검증한다. A가 B보다 작으면 불가능하다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  int a, b, x, y;
  cin >> a >> b;
  if (a < b)
    cout << "-1";
  else
  {
    x = (a + b) / 2;
    y = (a - b) / 2;
    if (x + y == a && x - y == b)
      cout << x << " " << y;
    else
      cout << "-1";
  }
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
