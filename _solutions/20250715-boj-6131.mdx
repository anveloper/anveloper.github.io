## 문제

[BOJ 6131 - 완전 제곱수](https://www.acmicpc.net/problem/6131)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
N = int(input())
cnt = 0
for i in range(1, N):
    if i**2 - (i - 1) ** 2 > N:
        break
    for j in range(1, i + 1):
        if i**2 - j**2 == N:
            cnt += 1
print(cnt)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
