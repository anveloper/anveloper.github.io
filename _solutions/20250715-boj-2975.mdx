## 문제

[BOJ 2975 - Transactions](https://www.acmicpc.net/problem/2975)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
while 1:
    a, op, b = input().split()
    if a == b == "0":
        break
    s = int(a) - int(b) if op == "W" else int(a) + int(b)
    print("Not allowed" if s < -200 else s)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
