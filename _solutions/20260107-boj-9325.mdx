## 문제

[BOJ 9325 - 얼마?](https://www.acmicpc.net/problem/9325)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);

  int t;
  cin >> t;

  while (t--)
  {
    int price, n;
    cin >> price >> n;

    while (n--)
    {
      int q, p;
      cin >> q >> p;

      price += q * p;
    }

    cout << price << '\n';
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
