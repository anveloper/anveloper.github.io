## 문제

[BOJ 2522 - 별 찍기 - 12](https://www.acmicpc.net/problem/2522)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```java
package day699;

import java.io.*;

public class Day652BOJ2522별찍기12 {
  public static void main(String[] args) throws Exception {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    int n = Integer.parseInt(br.readLine());
    StringBuilder sb = new StringBuilder();
    for (int i = 1; i <= n; i++) {
      for (int j = n - i; j > 0; j--)
        sb.append(" ");
      for (int j = 1; j <= i; j++)
        sb.append("*");
      sb.append("\n");
    }
    for (int i = 1; i <= n - 1; i++) {
      for (int j = 1; j <= i; j++)
        sb.append(" ");
      for (int j = 1; j <= n - i; j++)
        sb.append("*");
      sb.append("\n");
    }
    System.out.print(sb);
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
