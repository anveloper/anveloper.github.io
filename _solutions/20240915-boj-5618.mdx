## 문제

[BOJ 5618 - 공약수](https://www.acmicpc.net/problem/5618)

> 2개 또는 3개의 자연수가 주어질 때, 이들의 공약수를 오름차순으로 모두 출력한다.

## 풀이

유클리드 호제법으로 최대공약수(GCD)를 구한 뒤, 1부터 GCD까지 순회하며 GCD의 약수를 모두 출력한다. 3개의 수인 경우 두 수의 GCD를 먼저 구하고 그 결과와 세 번째 수의 GCD를 구한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int GCD(int a, int b)
{
  if (b == 0)
  {
    return a;
  }
  return GCD(b, a % b);
}
int main()
{
  int n, a, b, c, g;
  cin >> n;
  if (n == 2)
  {
    cin >> a >> b;
    g = GCD(a, b);
    for (int i = 1; i <= g; i++)
      if (g % i == 0)
        cout << i << '\n';
  }
  else
  {
    cin >> a >> b >> c;
    g = GCD(a, GCD(b, c));
    for (int i = 1; i <= g; i++)
      if (g % i == 0)
        cout << i << '\n';
  }
}
```

## 복잡도

- 시간: O(GCD(a, b))
- 공간: O(1)
