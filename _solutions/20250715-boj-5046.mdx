## 문제

[BOJ 5046 - 전국 대학생 프로그래밍 대회 동아리 연합](https://www.acmicpc.net/problem/5046)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
N, B, H, W = map(int, input().split())
cost = []
hotel = []
total = []
for i in range(H):
    cost.append(int(input()))
    hotel.append(list(map(int, input().split())))

for i in range(H):
    for j in hotel[i]:
        if j > N:
            total.append(N * cost[i])

if len(total) == 0:
    print("stay home")
elif min(total) > B:
    print("stay home")
else:
    print(min(total))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
