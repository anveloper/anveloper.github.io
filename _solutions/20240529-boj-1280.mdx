## 문제

[BOJ 1280 - 나무 심기](https://www.acmicpc.net/problem/1280)

> N개의 나무를 순서대로 심을 때, 2번째 나무부터 이전에 심은 모든 나무와의 거리 합을 구하고 그 곱을 출력한다.

## 풀이

각 테스트 케이스에서 N명의 이름과 2N-1개의 값을 입력받은 뒤, 값을 정렬하여 짝이 없는 유일한 값(홀수 번 등장)을 찾아 해당 인덱스의 이름을 출력한다. 정렬 후 인접한 값을 비교하여 쌍을 이루지 못하는 값을 식별한다.

## 코드

```python
n = 1
result = []

while n:
    n = int(input())
    names = [input() for i in range(n)]
    values = []

    for i in range(2 * n - 1):
        a = int(input().split()[0])
        values.append(a)

    values.sort()
    for i in range(0, len(values), 2):
        if i == (len(values) - 1) or values[i] != values[i + 1]:
            result.append(names[values[i] - 1])
            break

for i in range(len(result)):
    print(i + 1, result[i])
```

## 복잡도

- 시간: O(N log N)
- 공간: O(N)
