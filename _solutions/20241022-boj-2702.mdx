## 문제

[BOJ 2702 - 초6 수학](https://www.acmicpc.net/problem/2702)

> 두 자연수가 주어졌을 때, 최소공배수와 최대공약수를 출력한다.

## 풀이

유클리드 호제법으로 최대공약수(GCD)를 구한 뒤, 두 수의 곱을 GCD로 나눠 최소공배수(LCM)를 계산한다. 각 테스트 케이스마다 LCM과 GCD를 공백으로 구분하여 출력한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int gcd(int a, int b)
{
  int r;
  while (b != 0)
  {
    r = a % b;
    a = b;
    b = r;
  }
  return a;
}

int main()
{

  int n;
  cin >> n;

  int a, b;

  for (int i = 0; i < n; i++)
  {

    cin >> a >> b;

    cout << (a * b) / gcd(a, b) << " ";
    cout << gcd(a, b) << "\n";
  }
}
```

## 복잡도

- 시간: O(N * log(min(a, b)))
- 공간: O(1)
