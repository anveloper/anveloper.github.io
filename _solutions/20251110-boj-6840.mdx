## 문제

[BOJ 6840 - Who is in the middle?](https://www.acmicpc.net/problem/6840)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);

  int num[3];
  for (int i = 0; i < 3; i++)
  {
    cin >> num[i];
  }
  sort(num, num + 3);

  cout << num[1] << endl;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
