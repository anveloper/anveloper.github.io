## 문제

[BOJ 5013 - Death Knight Hero](https://www.acmicpc.net/problem/5013)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  cin >> n;

  int cntWins = 0;
  for (int i = 0; i < n; i++)
  {
    string abilities;
    cin >> abilities;

    bool hasCD = false;
    for (size_t j = 0; j < abilities.size(); j++)
    {
      if (abilities[j] == 'C' && abilities[j + 1] == 'D')
      {
        hasCD = true;
        break;
      }
    }

    if (!hasCD)
      cntWins++;
  }

  cout << cntWins << "\n";

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
