## 문제

[BOJ 3208 - gus](https://www.acmicpc.net/problem/3208)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int t(int a, int b)
{
  if (a && b)
    return t(b, a - 1) + 1;
  return -1;
}
int main()
{
  int m, n;
  cin >> m >> n;
  cout << t(m, n);
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
