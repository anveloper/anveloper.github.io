## 문제

[BOJ 10820 - 문자열 분석](https://www.acmicpc.net/problem/10820)

> 문자열이 주어질 때, 소문자, 대문자, 숫자, 공백의 개수를 각각 출력한다.

## 풀이

getline으로 한 줄씩 읽어 각 문자를 분류한다. 소문자, 대문자, 숫자, 공백 각각의 범위를 조건문으로 확인하여 개수를 세고 출력한다.

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
  string s;
  while (getline(cin, s))
  {
    int a = 0, b = 0, c = 0, d = 0;
    for (int i = 0; i < s.size(); i++)
    {
      if (s[i] >= 'a' && s[i] <= 'z')
        a++;
      else if (s[i] >= 'A' && s[i] <= 'Z')
        b++;
      else if (s[i] >= '0' && s[i] <= '9')
        c++;
      else if (s[i] == ' ')
        d++;
    }
    cout << a << " " << b << " " << c << " " << d << '\n';
  }
  return 0;
}
```

## 복잡도

- 시간: O(T * L) (T: 줄 수, L: 문자열 길이)
- 공간: O(L)
