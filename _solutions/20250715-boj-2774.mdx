## 문제

[BOJ 2774 - 아름다운 수](https://www.acmicpc.net/problem/2774)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
t = int(input())

for _ in range(t):
    x = input()
    data = []

    result = 0
    for i in range(len(x)):
        if int(x[i]) not in data:
            result += 1
            data.append(int(x[i]))

    print(result)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
