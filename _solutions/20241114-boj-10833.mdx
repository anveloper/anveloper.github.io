## 문제

[BOJ 10833 - 사과](https://www.acmicpc.net/problem/10833)

> 각 학교의 학생 수와 사과 수가 주어질 때, 사과를 학생 수로 나눈 나머지의 총합을 구한다.

## 풀이

각 학교마다 사과 수를 학생 수로 나눈 나머지를 구하고, 모든 학교의 나머지를 합산하여 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  int student, apple, num, sum = 0;
  cin >> num;
  for (int i = 0; i < num; i++)
  {
    cin >> student >> apple;
    sum += apple % student;
  }
  cout << sum << '\n';
}
```

## 복잡도

- 시간: O(N) (N: 학교 수)
- 공간: O(1)
