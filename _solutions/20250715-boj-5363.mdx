## 문제

[BOJ 5363 - 요다](https://www.acmicpc.net/problem/5363)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
n = int(input())

for _ in range(n):
    sentence = input().split()
    front = sentence[:2]
    rest = sentence[2:]
    new_sentence = rest + front
    print(" ".join(new_sentence))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
