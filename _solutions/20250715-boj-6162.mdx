## 문제

[BOJ 6162 - Superlatives](https://www.acmicpc.net/problem/6162)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.read
data = input().split()

K = int(data[0])
result = []

index = 1
for case_num in range(1, K + 1):
    E = int(data[index])
    A = int(data[index + 1])
    index += 2

    result.append(f"Data Set {case_num}:")

    if E <= A:
        result.append("no drought")
    else:
        ratio = E / A
        if ratio <= 5:
            result.append("drought")
        else:
            count = 0
            threshold = 5
            while ratio > threshold:
                count += 1
                threshold *= 5
            result.append("mega " * count + "drought")

    result.append("")  # blank line

print("\n".join(result))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
