## 문제

[BOJ 2097 - 조약돌](https://www.acmicpc.net/problem/2097)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
def solve(n):
    if n == 1 or n == 2:
        return 4
    else:
        temp = round((n) ** 0.5)
        if temp * temp >= n:
            return (temp - 1) * 4
        else:
            return ((temp - 1) * 2) + (temp * 2)


N = int(input())
print(solve(N))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
