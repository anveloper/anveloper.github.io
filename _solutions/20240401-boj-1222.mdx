## 문제

[BOJ 1222 - 홍준 프로그래밍 대회](https://www.acmicpc.net/problem/1222)

> N개의 학교에서 각 학교의 학생 수가 주어질 때, 같은 수의 배수인 학교들로 팀을 구성하여 최대 점수를 구한다.

## 풀이

카운팅 배열을 만들어 각 학생 수의 빈도를 기록한다. 2부터 최대값까지 모든 약수 d에 대해, d의 배수인 학교 수를 세고 d × 학교 수의 최대값을 갱신한다. 조화수 급수의 성질로 시간 내에 탐색 가능하다.

## 코드

```javascript
const solution = (input) => {
  const N = Number(input[0]);
  const arr = input[1].split(" ").map(Number);
  const M = Math.max(...arr);
  const carr = Array(M + 1).fill(0);
  arr.map((s) => carr[s]++);
  let ans = N;
  for (let d = 2; d <= M; d++) {
    let cnt = 0;
    for (let i = d; i <= M; i += d) cnt += carr[i];
    if (cnt > 1 && d * cnt > ans) ans = d * cnt;
  }
  return ans;
};

const fs = require("fs");
const input = fs.readFileSync("/dev/stdin").toString().trim().split("\n");
console.log(solution(input));
```

## 복잡도

- 시간: O(M log M) — M은 최대 학생 수
- 공간: O(M)
