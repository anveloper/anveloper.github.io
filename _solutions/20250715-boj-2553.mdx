## 문제

[BOJ 2553 - 마지막 팩토리얼 수](https://www.acmicpc.net/problem/2553)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n = int(input())
ans = 1
for i in range(1,n+1):
	ans *= i
ans = str(ans)
for i in range(len(ans)-1,-1,-1):
	
	if int(ans[i]) != 0:
		print(ans[i])
		break
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
