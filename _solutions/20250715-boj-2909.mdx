## 문제

[BOJ 2909 - 캔디 구매](https://www.acmicpc.net/problem/2909)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
c, k = map(int, input().split())
print(int(round(c + 0.1, -k)))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
