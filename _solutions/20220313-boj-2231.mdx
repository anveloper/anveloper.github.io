## 문제

[BOJ 2231 - 분해합](https://www.acmicpc.net/problem/2231)

## 풀이

가능한 모든 경우를 탐색하여 답을 구한다.

## 코드

```java
package com.ssafy.an.day049;

import java.util.Scanner;

public class Day42BOJ2231분해합브루트포스 { // 2231 분해합
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int N = sc.nextInt();
		int ans = 0;
		for (int i = 0; i < N; i++) {
			int tmp = i;
			int sum = 0;
			while (tmp != 0) {
				sum += tmp % 10;
				tmp /= 10;
			}
			if (sum + i == N) {
				ans = i;
				break;
			}
		}

		System.out.println(ans);
		sc.close();
	}
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
