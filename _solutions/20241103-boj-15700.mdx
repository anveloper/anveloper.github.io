## 문제

[BOJ 15700 - 타일 채우기 4](https://www.acmicpc.net/problem/15700)

> N x M 크기의 격자를 2x1 도미노 타일로 채울 때 최대 타일 개수를 구한다.

## 풀이

N*M 칸을 2칸씩 묶으면 되므로, 최대 타일 개수는 N*M/2이다. 정수 나눗셈으로 바로 계산한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int main()
{
  long x, y;
  cin >> x >> y;
  cout << x * y / 2;
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
