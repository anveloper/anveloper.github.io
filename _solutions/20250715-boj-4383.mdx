## 문제

[BOJ 4383 - 점프는 즐거워](https://www.acmicpc.net/problem/4383)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

input = sys.stdin.readline
while 1:
    try:
        data = list(map(int, input().split()))
        target = set([i for i in range(1, data[0])])
        answer = "Jolly"
        for i in range(1, data[0]):
            diff = abs(data[i] - data[i + 1])
            if diff not in target:
                answer = "Not jolly"
            else:
                target.remove(diff)
        if len(target) > 0:
            answer = "Not jolly"
        print(answer)
    except:
        break
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
