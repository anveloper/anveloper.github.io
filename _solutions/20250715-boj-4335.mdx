## 문제

[BOJ 4335 - 숫자 맞추기](https://www.acmicpc.net/problem/4335)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
import sys

lst_high = []
lst_low = []

while 1:
    num = int(sys.stdin.readline())
    if num == 0:
        break
    what = str(sys.stdin.readline().strip())
    if what == "too high":
        lst_high.append(num)
    elif what == "too low":
        lst_low.append(num)
    else:
        if len(lst_low) != 0 and len(lst_high) != 0:
            if max(lst_low) >= num or min(lst_high) <= num:
                print("Stan is dishonest")
            else:
                print("Stan may be honest")
        elif len(lst_low) != 0:
            if max(lst_low) >= num:
                print("Stan is dishonest")
            else:
                print("Stan may be honest")
        elif len(lst_high) != 0:
            if min(lst_high) <= num:
                print("Stan is dishonest")
            else:
                print("Stan may be honest")
        else:
            print("Stan may be honest")
        lst_high = []
        lst_low = []
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
