## 문제

[BOJ 2511 - 카드놀이](https://www.acmicpc.net/problem/2511)

> A와 B가 10라운드에 걸쳐 카드를 내고, 큰 수가 3점, 비기면 각 1점을 얻을 때 최종 점수와 승자를 출력한다.

## 풀이

10라운드를 순회하며 각 라운드의 승패에 따라 점수를 누적하고, 마지막으로 이긴 사람을 기록한다. 총점이 같을 때는 마지막 승리자가 최종 승자가 되며, 모두 비겼으면 D를 출력한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int a[10], b[10], as, bs;
char l = 'D';

int main()
{
  for (int &i : a)
    cin >> i;
  for (int &i : b)
    cin >> i;
  for (int i = 0; i < 10; ++i)
    a[i] == b[i] ? ++as, ++bs : a[i] > b[i] ? (as += 3, l = 'A')
                                            : (bs += 3, l = 'B');
  cout << as << " " << bs << "\n"
       << (as == bs ? l : as > bs ? 'A'
                                  : 'B');
}
```

## 복잡도

- 시간: O(1) (고정 10라운드)
- 공간: O(1)
