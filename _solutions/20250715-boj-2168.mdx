## 문제

[BOJ 2168 - 타일 위의 대각선](https://www.acmicpc.net/problem/2168)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import math

x, y = map(int, input().split())
a = math.gcd(x, y)
ans = (x // a + y // a - 1) * a
print(ans)
```

## 복잡도

- 시간: O(log N)
- 공간: O(1)
