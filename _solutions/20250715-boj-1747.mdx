## 문제

[BOJ 1747 - 소수&팰린드롬](https://www.acmicpc.net/problem/1747)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
n = int(input())

def isPrime(n):
    if n == 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def isPalindrome(n):
    if str(n) == str(n)[::-1]:
        return True
    return False

while True:
    if isPalindrome(n) and isPrime(n):
        break
    n += 1
print(n)
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
