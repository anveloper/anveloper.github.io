## 문제

[BOJ 8574 - Ratownik](https://www.acmicpc.net/problem/8574)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys

input = sys.stdin.readline
n, k, x, y = map(int, input().split())
ans = 0
for _ in range(n):
    xi, yi = map(int, input().split())
    if ((xi - x) ** 2 + (yi - y) ** 2) ** 0.5 > k:
        ans += 1
print(ans)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
