## 문제

[BOJ 13597 - Tri-du](https://www.acmicpc.net/problem/13597)

> 정삼각형의 두 변이 주어졌을 때 나머지 한 변의 길이를 출력한다.

## 풀이

정삼각형이므로 세 변의 길이가 모두 같다. 두 변 중 큰 값을 출력하면 된다.

## 코드

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main(void)
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);
  int A, B;
  cin >> A >> B;
  int result = max(A, B);
  cout << result << "\n";
  return 0;
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
