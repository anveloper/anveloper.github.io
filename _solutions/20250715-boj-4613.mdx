## 문제

[BOJ 4613 - Quicksum](https://www.acmicpc.net/problem/4613)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
def quicksum(s: str) -> int:
    total = 0
    for i, ch in enumerate(s, start=1):
        if ch == " ":
            continue
        value = ord(ch) - ord("A") + 1
        total += i * value
    return total


def main():
    while True:
        line = input()
        if line == "#":
            break
        print(quicksum(line))


if __name__ == "__main__":
    main()
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
