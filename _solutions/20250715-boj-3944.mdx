## 문제

[BOJ 3944 - 나머지 계산](https://www.acmicpc.net/problem/3944)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
t = int(input())

for _ in range(t):
    b, d = input().split()
    b = int(b)
    tmp = sum(map(int, list(d)))
    print(tmp % (b - 1))
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
