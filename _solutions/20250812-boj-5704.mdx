## 문제

[BOJ 5704 - 팬그램](https://www.acmicpc.net/problem/5704)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <string>
#include <bitset>

using namespace std;

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  while (getline(cin, s) && s != "*")
  {
    bitset<26> seen;
    for (char c : s)
      if (isalpha(c))
        seen[c - 'a'] = 1;
    cout << (seen.all() ? "Y" : "N") << "\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
