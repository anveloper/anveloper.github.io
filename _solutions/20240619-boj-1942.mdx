## 문제

[BOJ 1942 - 디지털시계](https://www.acmicpc.net/problem/1942)

> 시작 시각부터 종료 시각까지 매초마다 시분초를 이어 붙인 6자리 수가 3의 배수인 횟수를 구한다.

## 풀이

시각을 초 단위로 변환하여 시작부터 종료까지 순회한다. 각 시각을 HHMMSS 형태의 수로 변환한 뒤 3으로 나눠지는지 확인한다. 종료 시각이 시작보다 작으면 자정을 넘기는 것이므로 24시간을 더해 처리한다.

## 코드

```python
def realtime(h, m, s):
    return 3600 * h + 60 * m + s


def tentime(val):
    return 10000 * (val // 3600) + 100 * ((val % 3600) // 60) + val % 60


for i in range(3):
    a, b = input().split()
    answer = 0
    dataa = list(map(int, a.split(":")))
    datab = list(map(int, b.split(":")))
    vala = realtime(*dataa)
    valb = realtime(*datab)
    if vala > valb:
        valb += realtime(24, 0, 0)
    for i in range(vala, valb + 1):
        if tentime(i) % 3 == 0:
            answer += 1
    print(answer)
```

## 복잡도

- 시간: O(86400) = O(1) (최대 하루 초 수)
- 공간: O(1)
