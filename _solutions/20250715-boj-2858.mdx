## 문제

[BOJ 2858 - 기숙사 바닥](https://www.acmicpc.net/problem/2858)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
R, B = map(int, input().split())
LpW = (R + 4) // 2
LW = R + B

for i in range(1, LpW):
    if (LpW - i) * i == LW:
        print(max(i, LpW - i), min(i, LpW - i))
        break
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
