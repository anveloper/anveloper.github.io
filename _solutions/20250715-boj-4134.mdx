## 문제

[BOJ 4134 - 다음 소수](https://www.acmicpc.net/problem/4134)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```python
import sys

input = sys.stdin.readline
N = int(input())


def check(x):
    for i in range(2, int(x**0.5) + 1):
        if x % i == 0:
            return False
    return True


for i in range(N):
    n = int(input())
    while True:
        if n == 0 or n == 1:
            print(2)
            break
        if check(n):
            print(n)
            break
        else:
            n += 1
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
