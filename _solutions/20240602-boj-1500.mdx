## 문제

[BOJ 1500 - 최대 곱](https://www.acmicpc.net/problem/1500)

> 합이 S인 M개의 자연수의 곱을 최대로 만드는 값을 구한다.

## 풀이

S를 M으로 나눈 몫 p와 나머지 q를 구한다. 곱을 최대화하려면 수들의 차이를 최소화해야 하므로, q개의 수는 (p+1)로, 나머지 (M-q)개의 수는 p로 설정하여 곱을 계산한다.

## 코드

```python
import sys

n, m = map(int, sys.stdin.readline().split(" "))
p = n // m
q = n % m
ans = 1
for _ in range(m):
    if q > 0:
        ans *= p + 1
        q -= 1
    else:
        ans *= p
print(ans)
```

## 복잡도

- 시간: O(M)
- 공간: O(1)
