## 문제

[BOJ 2033 - 반올림](https://www.acmicpc.net/problem/2033)

> 일의 자리에서부터 시작하여 반올림을 반복하되, 해당 자리가 0이 아닌 동안 계속 반올림한다.

## 풀이

자릿수를 10부터 시작하여 키워가며, 현재 자릿수의 값이 절반 이상이면 올림, 아니면 버림 처리한다. 결과가 현재 단위 이상인 동안 반복한다.

## 코드

```cpp
#include <iostream>
using namespace std;
int main()
{
  long long n;
  cin >> n;

  long long tmp = 10;

  while (n >= tmp)
  {
    (n % tmp) >= tmp / 2 ? n = ((long long)((int)n / tmp) + 1) *tmp : n = ((long long)(int)n / tmp) * tmp;
    tmp *= 10;
  }
  cout << n << endl;
}
```

## 복잡도

- 시간: O(log N)
- 공간: O(1)
