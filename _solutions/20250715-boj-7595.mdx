## 문제

[BOJ 7595 - Triangles](https://www.acmicpc.net/problem/7595)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
while 1:
    n = int(input())
    if n == 0:
        break
    for i in range(1, n + 1):
        print("*" * i)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
