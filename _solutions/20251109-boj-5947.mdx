## 문제

[BOJ 5947 - Book Club](https://www.acmicpc.net/problem/5947)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
using namespace std;

int n, q, p, a[50001][51], check[50001];

int main()
{
  ios_base::sync_with_stdio(0);
  cin.tie(0);

  cin >> n >> q >> p;
  for (int i = 1; i <= n; i++)
    for (int j = 1; j <= q; j++)
      cin >> a[i][j];

  for (int i = 0; i < p; i++)
  {
    int qj, aj;
    cin >> qj >> aj;
    for (int j = 1; j <= n; j++)
    {
      if (a[j][qj] == aj)
        check[j]++;
    }
  }

  int ans = 0;
  for (int i = 1; i <= n; i++)
  {
    if (check[i] == p)
      ans++;
  }
  cout << ans << '\n';
  return 0;
}
```

## 복잡도

- 시간: O(N²)
- 공간: O(N)
