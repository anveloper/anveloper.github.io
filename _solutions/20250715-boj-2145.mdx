## 문제

[BOJ 2145 - 숫자 놀이](https://www.acmicpc.net/problem/2145)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```python
while(True):
    n = int(input())
    if n == 0:
        break
    while(n>9):
        n = sum(map(int,list(str(n))))
    print(n)
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
