## 문제

[BOJ 4072 - Words](https://www.acmicpc.net/problem/4072)

## 풀이

문제의 조건을 그대로 구현한다.

## 코드

```cpp
#include <iostream>
#include <cstring>

int main()
{
  char str[251], c;
  int len;
  while (1)
  {
    scanf("%s", str);
    if (str[0] == '#' && str[1] == NULL)
      break;
    len = strlen(str);
    for (int i = len - 1; i >= 0; i--)
    {
      printf("%c", str[i]);
    }
    c = getc(stdin);
    printf("%c", c);
  }
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
