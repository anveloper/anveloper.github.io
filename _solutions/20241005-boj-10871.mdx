## 문제

[BOJ 10871 - X보다 작은 수](https://www.acmicpc.net/problem/10871)

> N개의 정수 중에서 X보다 작은 수를 입력받은 순서대로 출력한다.

## 풀이

N개의 수를 배열에 입력받은 뒤, 각 원소가 X보다 작은지 비교하여 해당하는 수만 출력한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
  int N, X;
  cin >> N >> X;

  int arr[10000];
  for (int i = 0; i < N; i++)
  {
    cin >> arr[i];
  }

  for (int i = 0; i < N; i++)
  {
    if (arr[i] < X)
    {
      cout << arr[i] << " ";
    }
  }
  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
