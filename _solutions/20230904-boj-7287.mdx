## 문제

[BOJ 7287 - 등록](https://www.acmicpc.net/problem/7287)

자신이 백준 온라인 저지에서 맞은 문제 수와 아이디를 출력한다.

### 입력

입력 없음.

### 출력

첫째 줄에 맞은 문제 수, 둘째 줄에 아이디를 출력한다.

### 예제

| 입력   | 출력            |
| ------ | --------------- |
| (없음) | `703` `hitedin` |

## 풀이

하드코딩된 문자열을 출력한다.

1. 맞은 문제 수와 아이디를 println으로 출력한다

**핵심 아이디어**: 입출력 연습용 문제로, 고정된 값을 출력하면 된다.

## 코드

```java
package day599;

public class Day577BOJ7287등록 {
  public static void main(String[] args) {
    System.out.println("703\nhitedin");
  }
}
```

## 복잡도

- 시간: O(1)
- 공간: O(1)
