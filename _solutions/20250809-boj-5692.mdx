## 문제

[BOJ 5692 - 팩토리얼 진법](https://www.acmicpc.net/problem/5692)

## 풀이

수학적 성질을 이용하여 풀이한다.

## 코드

```cpp
#include <iostream>

using namespace std;

int N;
int arr[6];

bool init()
{

  cin >> N;
  return N;
}

int main()
{
  ios::sync_with_stdio(false);
  cin.tie(NULL);
  for (int i = 0; i < 5; i++)
  {
    if (i == 0)
      arr[i] = 1;
    else
      arr[i] = arr[i - 1] * (i + 1);
  }
  while (init())
  {
    int temp = N;
    int j = 0;
    int sum = 0;
    while ((temp % 10) || (temp / 10))
    {
      sum += (temp % 10) * arr[j++];
      temp /= 10;
    }
    cout << sum << "\n";
  }

  return 0;
}
```

## 복잡도

- 시간: O(N)
- 공간: O(N)
